Masterclass 1: Fundamentos de HTML

El Esqueleto de la Web

HTML (HyperText Markup Language) es el lenguaje que usamos para estructurar el contenido web.

index.html
<!DOCTYPE html>
<html>
<head>
    <title>Mi Primera Web</title>
</head>
<body>
    <h1>Hola Universo</h1>
    <p>Esta es mi primera página web usando HTML.</p>
    <img src="foto.jpg" alt="Una foto genial">
</body>
</html>

Etiquetas Principales:

  • <h1> a <h6>: Títulos principales y subtítulos.
  • <p>: Párrafos de texto.
  • <img>: Imágenes (No requiere etiqueta de cierre).
  • <a>: Enlaces a otras páginas.

Masterclass 3: CSS Básico

Diseño y Estilo

CSS es la "ropa" y el "maquillaje" de HTML. Te permite dar color, tamaño y posición.

style.css
/* Selector de etiqueta */
body {
    background-color: #1a1a2e;
    color: #ffffff;
    font-family: Arial, sans-serif;
}

/* Selector de clase (punto) */
.boton-cool {
    background: #e94560;
    padding: 10px 20px;
    border-radius: 8px;
}

/* Selector de ID (hashtag) */
#titulo-principal {
    font-size: 3rem;
    text-align: center;
}

Masterclass 4: CSS Avanzado (Layouts)

Flexbox y el Modelo de Caja

Acomodar elementos en la pantalla puede ser un dolor de cabeza, ¡hasta que conoces Flexbox!

layout.css
.contenedor-flex {
    display: flex;
    justify-content: space-between; /* Separa los elementos */
    align-items: center; /* Los centra verticalmente */
    gap: 20px; /* Espacio entre ellos */
}

/* El modelo de caja */
.caja {
    width: 200px;
    padding: 20px; /* Espacio interno */
    margin: 15px; /* Espacio externo (aleja de otros) */
    border: 2px solid cyan;
}

Masterclass 5: Lógica con JavaScript

El Cerebro de la Web

JS nos permite hacer operaciones, tomar decisiones (if/else) y repetir tareas (ciclos).

logica.js
// Variables
let vidas = 3;
const nivel = "Experto";

// Condicionales (Tomar decisiones)
if (vidas > 0) {
    console.log("¡Sigue jugando!");
} else {
    console.log("Game Over");
}

// Ciclos (Repetir cosas)
for(let i = 1; i <= 5; i++) {
    console.log(`Disparo láser #${i}`);
}

Masterclass 6: Interacción y el DOM

Controlando la Pantalla

El DOM (Document Object Model) es la representación en JS de todo tu HTML. ¡Puedes modificar la página en vivo!

interactividad.js
// 1. Seleccionar un elemento HTML
const boton = document.getElementById("miBoton");
const titulo = document.querySelector(".titulo-principal");

// 2. Agregar un "escuchador de eventos" (Click)
boton.addEventListener("click", function() {
    // 3. Modificar el HTML y CSS en vivo!
    titulo.textContent = "¡Has hackeado la web!";
    titulo.style.color = "lime";
    
    // Crear una alerta
    alert("¡Botón presionado!");
});

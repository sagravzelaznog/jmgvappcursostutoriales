Gráficos 3D en GeoGebra

GeoGebra te permite crear y manipular objetos en tres dimensiones, abriendo un mundo de posibilidades para visualizar conceptos matemáticos complejos.

Activando la Vista 3D

  1. Ve al menú Vista > Vista 3D
  2. O usa el atajo Ctrl+Shift+3 (Windows/Linux) o Cmd+Shift+3 (Mac)
Ejemplo: Crear un Cono 3D
  1. Abre la Vista 3D
  2. Selecciona la herramienta Cono
  3. Haz clic para establecer el centro de la base
  4. Arrastra para definir el radio de la base
  5. Haz clic nuevamente para establecer la altura

Navegación en 3D

  • Rotar vista: Haz clic y arrastra con el botón izquierdo
  • Mover vista: Haz clic y arrastra con el botón derecho
  • Zoom: Usa la rueda del ratón o gesto de pellizco

Scripting en GeoGebra

El scripting te permite automatizar tareas y crear construcciones dinámicas más complejas.

Tipos de Scripts

  • Al hacer clic: Se ejecutan al hacer clic en un objeto
  • Al actualizar: Se ejecutan cuando cambia el valor de un objeto
  • Global JavaScript: Funciones reutilizables en toda la construcción
Ejemplo: Animación con Script

Crea una animación que haga girar un polígono regular.

  1. Crea un deslizador 'ángulo' de 0° a 360°
  2. Crea un polígono regular
  3. Haz clic derecho en el deslizador > Propiedades > Scripting
  4. En 'Al actualizar', escribe: Gira(pol1, ángulo, A) (donde A es el centro de rotación)
  5. Activa la animación del deslizador
// Script de animación
function onUpdate() {
    // Obtener el valor actual del ángulo
    var angulo = getValue('ángulo');
    // Rotar el polígono
    ggbApplet.evalCommand('Gira(pol1, ' + angulo + ', A)');
}

Estadística Avanzada

GeoGebra incluye potentes herramientas para análisis estadístico avanzado.

Distribuciones de Probabilidad

Puedes trabajar con diversas distribuciones de probabilidad:

Distribución Sintaxis Ejemplo
Normal Normal(μ, σ, x) Normal(0, 1, x)
Binomial Binomial(n, p, k) Binomial(10, 0.5, 3)
Poisson Poisson(λ, k) Poisson(5, 3)
Ejemplo: Ajuste de Curvas

Ajusta una curva a un conjunto de datos.

  1. Ingresa los datos en la hoja de cálculo (Vista > Hoja de Cálculo)
  2. Selecciona las celdas con los datos
  3. Haz clic en el menú Análisis > Ajuste de Curva
  4. Selecciona el tipo de función (lineal, polinómica, exponencial, etc.)
  5. Haz clic en Aceptar

Proyectos Avanzados

Pon a prueba tus habilidades con estos proyectos desafiantes.

Proyecto 1: Superficie Paramétrica

Crea una superficie paramétrica en 3D, como un toro o una esfera.

Para crear un toro:

Superficie(
    (4 + cos(u)) * cos(v),
    (4 + cos(u)) * sin(v),
    sin(u),
    u, 0, 2π,
    v, 0, 2π
)

Ajusta los parámetros para modificar el tamaño y la forma del toro.

Proyecto 2: Simulación Física

Crea una simulación de un péndulo simple con amortiguación.

  1. Crea un punto A como punto de anclaje
  2. Crea un punto B que represente la masa
  3. Usa un deslizador para el ángulo inicial
  4. Crea un segmento entre A y B
  5. Usa un script para simular el movimiento del péndulo

Fórmula del péndulo amortiguado:

θ''(t) = -g/L * sin(θ) - b/m * θ'

Donde g es la gravedad, L es la longitud, b es el coeficiente de amortiguación, y m es la masa.

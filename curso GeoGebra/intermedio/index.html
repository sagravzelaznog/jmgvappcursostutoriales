Derivadas en GeoGebra

Las derivadas son fundamentales en el cálculo diferencial y GeoGebra ofrece herramientas poderosas para visualizarlas.

Calcular la derivada de una función

Puedes calcular la derivada de una función usando la barra de entrada o el comando correspondiente.

Ejemplo: Derivada de una función cuadrática
  1. Ingresa la función en la barra de entrada: f(x) = x^2 + 3x + 2
  2. Para calcular su derivada, escribe: f'(x) o Derivada(f)
  3. Presiona Enter para ver la función derivada

La derivada de f(x) = x² + 3x + 2 es f'(x) = 2x + 3

Interpretación geométrica

La derivada en un punto representa la pendiente de la recta tangente a la curva en ese punto.

Consejo: Usa el comando RectaTangente(f, a) para dibujar la recta tangente a f en x = a.

Integrales en GeoGebra

GeoGebra permite calcular integrales definidas e indefinidas, así como visualizar áreas bajo curvas.

Calcular una integral definida

Para calcular el área bajo una curva entre dos puntos:

Ejemplo: Área bajo la curva

Calcular el área bajo la curva f(x) = x² entre x = 0 y x = 2

  1. Ingresa la función: f(x) = x^2
  2. Para la integral definida, escribe: Integral(f, 0, 2)
  3. Para resaltar el área, usa: Integral(f, 0, 2, false, true)

El área bajo la curva de 0 a 2 es igual a 8/3 ≈ 2.666...

Trabajando con Deslizadores

Los deslizadores permiten crear construcciones dinámicas donde puedes variar parámetros y ver cambios en tiempo real.

Crear un deslizador

  1. Haz clic en el botón Deslizador en la barra de herramientas
  2. Haz clic en la vista gráfica para colocar el deslizador
  3. En el cuadro de diálogo, configura el nombre, valor mínimo, máximo e incremento
Ejemplo: Función con parámetro ajustable

Crea una función lineal cuya pendiente puedas ajustar con un deslizador.

  1. Crea un deslizador llamado 'm' con valores de -5 a 5
  2. En la barra de entrada, escribe: f(x) = m * x
  3. Mueve el deslizador para ver cómo cambia la pendiente

Ejercicios Prácticos

Practica los conceptos aprendidos con estos ejercicios. Intenta resolverlos por tu cuenta antes de ver la solución.

Ejercicio 1: Puntos Críticos

Dada la función f(x) = x³ - 3x², encuentra sus puntos críticos.

  1. Ingresa la función: f(x) = x^3 - 3x^2
  2. Calcula la primera derivada: f'(x) = Derivada(f)
  3. Encuentra las raíces de f'(x): Raíces(f')
  4. Los puntos críticos están en x = 0 y x = 2
Ejercicio 2: Área entre curvas

Calcula el área entre las curvas f(x) = x² y g(x) = √x en el intervalo [0,1].

  1. Ingresa ambas funciones: f(x) = x^2 y g(x) = sqrt(x)
  2. Encuentra los puntos de intersección: Intersección(f, g, 0, 1)
  3. Calcula el área entre las curvas: Integral(g-f, 0, 1)
  4. El área es igual a 1/3

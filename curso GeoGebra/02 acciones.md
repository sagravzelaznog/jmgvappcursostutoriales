# Plan de Acción: Curso de GeoGebra

## Estructura del Sitio Web
- `index.html` - Página principal con introducción al curso
- `basico/` - Directorio para el nivel básico
  - `index.html` - Introducción al nivel básico
  - `graficos-2d.html` - Gráficos 2D
  - `tablas-valores.html` - Generación de tablas de valores
  - `ecuaciones-basicas.html` - Resolución de ecuaciones
- `intermedio/` - Directorio para el nivel intermedio
  - `index.html` - Introducción al nivel intermedio
  - `derivadas.html` - Cálculo de derivadas
  - `integrales.html` - Cálculo de integrales
  - `transformaciones.html` - Transformaciones con deslizadores
- `avanzado/` - Directorio para el nivel avanzado
  - `index.html` - Introducción al nivel avanzado
  - `graficos-3d.html` - Gráficos 3D
  - `estadistica.html` - Estadística y probabilidad
  - `construcciones-avanzadas.html` - Construcciones avanzadas
- `recursos/` - Directorio para recursos compartidos
  - `css/` - Estilos CSS
  - `js/` - Scripts JavaScript
  - `img/` - Imágenes y capturas de pantalla
  - `ejercicios/` - Archivos .ggb de ejercicios

## Niveles de Aprendizaje

### Nivel Básico
1. **Gráficos 2D**
   - Creación de funciones básicas
   - Personalización de ejes y cuadrícula
   - Uso de herramientas de dibujo
   - *Ejercicio práctico:* Graficar y analizar una función cuadrática

2. **Tablas de Valores**
   - Generación de tablas
   - Relación entre tablas y gráficos
   - Exportación de datos
   - *Ejercicio práctico:* Crear tabla de valores para una función lineal

3. **Ecuaciones Básicas**
   - Resolución gráfica
   - Resolución numérica
   - Intersección de funciones
   - *Ejercicio práctico:* Resolver un sistema de ecuaciones lineales

### Nivel Intermedio
1. **Derivadas**
   - Cálculo de derivadas
   - Interpretación geométrica
   - Puntos críticos y extremos
   - *Ejercicio práctico:* Analizar una función cúbica

2. **Integrales**
   - Cálculo de integrales definidas
   - Áreas bajo curvas
   - Aplicaciones prácticas
   - *Ejercicio práctico:* Calcular el área entre dos curvas

3. **Transformaciones**
   - Uso de deslizadores
   - Animaciones
   - Parámetros dinámicos
   - *Ejercicio práctico:* Crear una función con parámetros ajustables

### Nivel Avanzado
1. **Gráficos 3D**
   - Superficies 3D
   - Curvas en el espacio
   - Vistas y perspectivas
   - *Ejercicio práctico:* Graficar un paraboloide

2. **Estadística**
   - Distribuciones de probabilidad
   - Análisis de datos
   - Regresión
   - *Ejercicio práctico:* Análisis de datos de regresión lineal

3. **Construcciones Avanzadas**
   - Macros
   - Scripting
   - Aplicaciones interactivas
   - *Ejercicio práctico:* Crear una herramienta personalizada

Introducción a SVM

Las Máquinas de Vectores de Soporte (SVM) son algoritmos supervisados utilizados para clasificación y regresión. Buscan el mejor hiperplano de separación entre clases.

Cargando visualización de SVM...

Tipos de Kernels

Kernel Lineal

K(x, y) = x·y

Ideal para datos linealmente separables.

Kernel RBF (Gaussiano)

K(x, y) = exp(-γ||x - y||²)

Para datos no lineales. Ajuste con γ.

Implementación en Python

from sklearn import svm
from sklearn.datasets import make_classification
from sklearn.model_selection import train_test_split

# Generar datos de ejemplo
X, y = make_classification(n_samples=100, n_features=2, n_redundant=0, random_state=42)
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# Crear y entrenar el modelo
clf = svm.SVC(kernel='rbf', C=1.0)
clf.fit(X_train, y_train)

# Evaluar el modelo
accuracy = clf.score(X_test, y_test)
print(f"Precisión: {accuracy:.2f}")

# Vectores de soporte
print(f"Número de vectores de soporte: {len(clf.support_)}")

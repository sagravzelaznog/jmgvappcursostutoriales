Contenido Teórico

1. Introducción a la Optimización

La optimización es el proceso de encontrar la mejor solución (óptima) dentro de un conjunto de soluciones factibles. En el contexto de IA y machine learning, la optimización es fundamental para entrenar modelos y ajustar parámetros.

Problema General de Optimización

Dada una función objetivo \( f(x) \) y un conjunto de restricciones \( g_i(x) \leq 0 \), \( h_j(x) = 0 \), encontrar:

\[ \min_{x \in \mathbb{R}^n} f(x) \] \[ \text{sujeto a:} \] \[ g_i(x) \leq 0, \quad i = 1, \ldots, m \] \[ h_j(x) = 0, \quad j = 1, \ldots, p \]

2. Tipos de Problemas de Optimización

2.1 Optimización Lineal

La función objetivo y las restricciones son lineales.

\[ \min_{x} c^T x \] \[ \text{s.a.} \quad Ax \leq b \] \[ \quad \quad \; x \geq 0 \]

2.2 Optimización No Lineal

Al menos una de las funciones (objetivo o restricciones) no es lineal.

2.3 Optimización Convexa

La función objetivo es convexa y el conjunto factible es un conjunto convexo.

2.4 Optimización Entera

Las variables de decisión deben tomar valores enteros.

3. Conceptos Clave en Optimización

3.1 Conjuntos Convexos

Un conjunto \( C \) es convexo si para todo \( x, y \in C \) y \( \theta \in [0,1] \), se cumple que \( \theta x + (1-\theta)y \in C \).

3.2 Funciones Convexas

Una función \( f: \mathbb{R}^n \rightarrow \mathbb{R} \) es convexa si su dominio es un conjunto convexo y para todo \( x, y \) en el dominio y \( \theta \in [0,1] \):

\[ f(\theta x + (1-\theta)y) \leq \theta f(x) + (1-\theta)f(y) \]

3.3 Condiciones de Optimalidad

Para una función diferenciable \( f \), un punto \( x^* \) es un mínimo local si:

\[ \nabla f(x^*) = 0 \] \[ \nabla^2 f(x^*) \succeq 0 \; \text{(matriz hessiana semidefinida positiva)}\]

4. Métodos de Optimización

4.1 Método del Gradiente Descendente

Algoritmo iterativo para encontrar un mínimo local de una función diferenciable.

\[ x_{k+1} = x_k - \alpha \nabla f(x_k) \]

Donde \( \alpha \) es la tasa de aprendizaje.

4.2 Método de Newton

Utiliza información de segundo orden (Hessiana) para converger más rápido.

\[ x_{k+1} = x_k - [\nabla^2 f(x_k)]^{-1} \nabla f(x_k) \]

4.3 Descenso de Gradiente Estocástico (SGD)

Variante que usa un subconjunto aleatorio de datos en cada iteración, muy usado en aprendizaje profundo.

1. Implementación del Gradiente Descendente

Vamos a implementar el método del gradiente descendente para minimizar una función cuadrática.

import numpy as np
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D
from matplotlib import cm

def gradiente_descendente(f, grad_f, x0, alpha=0.1, max_iter=100, tol=1e-6):
    """
    Implementación del método del gradiente descendente.
    
    Parámetros:
    f: función objetivo
    grad_f: gradiente de la función objetivo
    x0: punto inicial
    alpha: tasa de aprendizaje
    max_iter: número máximo de iteraciones
    tol: tolerancia para la convergencia
    
    Retorna:
    x: punto óptimo
    history: historial de puntos visitados
    """
    x = np.array(x0, dtype=float)
    history = [x.copy()]
    
    for i in range(max_iter):
        # Calcular el gradiente
        grad = grad_f(x)
        
        # Actualizar los parámetros
        x_new = x - alpha * grad
        history.append(x_new.copy())
        
        # Verificar convergencia
        if np.linalg.norm(x_new - x) < tol:
            print(f"Convergencia alcanzada en la iteración {i+1}")
            break
            
        x = x_new
    else:
        print("Se alcanzó el número máximo de iteraciones")
    
    return x, np.array(history)

2. Optimización con Restricciones: Método del Lagrangiano Aumentado

Implementación del método del Lagrangiano Aumentado para resolver problemas con restricciones de igualdad.

import numpy as np
import matplotlib.pyplot as plt

def lagrangiano_aumentado(f, grad_f, h, grad_h, x0, rho=1.0, mu=10.0, max_iter=100, tol=1e-6):
    """
    Método del Lagrangiano Aumentado para optimización con restricciones de igualdad.
    
    Parámetros:
    f: función objetivo
    grad_f: gradiente de la función objetivo
    h: función de restricción (debe devolver un vector)
    grad_h: gradiente de la restricción (debe devolver una matriz jacobiana)
    x0: punto inicial
    rho: parámetro de penalización inicial
    mu: factor de aumento de la penalización
    max_iter: número máximo de iteraciones
    tol: tolerancia para la convergencia
    
    Retorna:
    x: punto óptimo
    history: historial de puntos visitados
    """
    x = np.array(x0, dtype=float)
    lambda_ = np.zeros_like(h(x))  # Multiplicadores de Lagrange
    history = [x.copy()]
    
    for k in range(max_iter):
        # Definir la función aumentada
        def lagrangiano_aum(x):
            h_val = h(x)
            return f(x) + np.dot(lambda_, h_val) + (rho/2) * np.sum(h_val**2)
        
        # Gradiente del lagrangiano aumentado
        def grad_lagrangiano_aum(x):
            h_val = h(x)
            return grad_f(x) + grad_h(x).T @ (lambda_ + rho * h_val)
        
        # Minimizar el lagrangiano aumentado (usamos gradiente descendente)
        x_new, _ = gradiente_descendente(
            lagrangiano_aum, 
            grad_lagrangiano_aum, 
            x, 
            alpha=0.01, 
            max_iter=1000, 
            tol=1e-6
        )
        
        history.append(x_new.copy())
        
        # Verificar convergencia
        if np.linalg.norm(x_new - x) < tol and np.linalg.norm(h(x_new)) < tol:
            print(f"Convergencia alcanzada en la iteración {k+1}")
            break
        
        # Actualizar multiplicadores de Lagrange y parámetro de penalización
        h_new = h(x_new)
        lambda_ = lambda_ + rho * h_new
        rho = mu * rho
        
        x = x_new
    else:
        print("Se alcanzó el número máximo de iteraciones")
    
    return x, np.array(history)

# Ejemplo: Minimizar f(x,y) = x^2 + y^2 sujeto a x + y = 1
def f_ejemplo(x):

# Punto inicial

    f_ejemplo, grad_f_ejemplo,

# Visualización
x = np.linspace(-5, 5, 100)
y = np.linspace(-5, 5, 100)

3. Aplicación: Regresión Lineal con Gradiente Descendente

Implementación de regresión lineal utilizando gradiente descendente.

import numpy as np
import matplotlib.pyplot as plt
from sklearn.datasets import make_regression
from sklearn.preprocessing import StandardScaler
from sklearn.model_selection import train_test_split

class RegresionLinealGD:
    def __init__(self, learning_rate=0.01, n_iter=1000, tol=1e-6):
        self.learning_rate = learning_rate
        self.n_iter = n_iter
        self.tol = tol
        self.theta = None
        self.loss_history = []
    
    def compute_loss(self, X, y):
        """Función de pérdida (Error Cuadrático Medio)"""
        m = len(y)
        predictions = X @ self.theta
        error = predictions - y
        return (1/(2*m)) * np.sum(error**2)
    
    def fit(self, X, y):
        """Entrenar el modelo con gradiente descendente"""
        # Añadir término de sesgo (columna de unos)
        X_b = np.c_[np.ones((X.shape[0], 1)), X]
        m = len(y)
        
        # Inicializar parámetros
        self.theta = np.random.randn(X_b.shape[1], 1)
        
        for i in range(self.n_iter):
            # Calcular predicciones
            predictions = X_b @ self.theta
            
            # Calcular gradiente
            gradients = (1/m) * X_b.T @ (predictions - y)
            
            # Actualizar parámetros
            self.theta = self.theta - self.learning_rate * gradients
            
            # Calcular y guardar la pérdida
            loss = self.compute_loss(X_b, y)
            self.loss_history.append(loss)
            
            # Verificar convergencia
            if i > 0 and abs(self.loss_history[-2] - self.loss_history[-1]) < self.tol:
                print(f"Convergencia alcanzada en la iteración {i+1}")
                break
        
        return self
    
    def predict(self, X):
        """Realizar predicciones"""
        X_b = np.c_[np.ones((X.shape[0], 1)), X]
        return X_b @ self.theta

# Generar datos de ejemplo
np.random.seed(42)
X, y = make_regression(n_samples=100, n_features=1, noise=20, random_state=42)
y = y.reshape(-1, 1)  # Convertir a vector columna

# Estandarizar características
scaler = StandardScaler()

# Dividir en conjuntos de entrenamiento y prueba
X_train, X_test, y_train, y_test = train_test_split(X_scaled, y, test_size=0.2, random_state=42)

# Crear y entrenar el modelo
modelo = RegresionLinealGD(learning_rate=0.1, n_iter=1000)
modelo.fit(X_train, y_train)

# Hacer predicciones
y_pred = modelo.predict(X_test)

# Calcular error cuadrático medio en el conjunto de prueba
mse = np.mean((y_pred - y_test)**2)
print(f"Error Cuadrático Medio (MSE) en prueba: {mse:.4f}")

# Visualización de resultados
plt.figure(figsize=(15, 5))

# Gráfico 1: Datos y línea de regresión
plt.subplot(1, 2, 1)
plt.scatter(X_train, y_train, color='blue', alpha=0.5, label='Datos de entrenamiento')

import numpy as np
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D

def himmelblau(x, y):
    """Función de Himmelblau"""
    return (x**2 + y - 11)**2 + (x + y**2 - 7)**2

def grad_himmelblau(x, y):
    """Gradiente de la función de Himmelblau"""
    df_dx = 4 * x * (x**2 + y - 11) + 2 * (x + y**2 - 7)
    df_dy = 2 * (x**2 + y - 11) + 4 * y * (x + y**2 - 7)
    return np.array([df_dx, df_dy])

def gradiente_descendente_himmelblau(x0, y0, alpha=0.01, max_iter=1000, tol=1e-6):
    """Gradiente descendente para la función de Himmelblau"""
    x, y = x0, y0
    history = [(x, y)]
    
    for i in range(max_iter):
        # Calcular gradiente
        grad = grad_himmelblau(x, y)
        
        # Actualizar parámetros
        x_new = x - alpha * grad[0]
        y_new = y - alpha * grad[1]
        
        history.append((x_new, y_new))
        
        # Verificar convergencia
        if np.sqrt((x_new - x)**2 + (y_new - y)**2) < tol:
            print(f"Convergencia alcanzada en la iteración {i+1}")
            break
            
        x, y = x_new, y_new
    else:
        print("Se alcanzó el número máximo de iteraciones")
    
    return np.array(history)

Ejercicio 2: Optimización con Restricciones de Desigualdad

Implementa el método de Barrera Logarítmica (Interior Point) para resolver el siguiente problema de optimización:

\[ \min_{x, y} (x - 4)^2 + (y - 4)^2 \] \[ \text{sujeto a:} \] \[ x + y \leq 5 \] \[ x \geq 0, y \geq 0 \]

Tareas:

  1. Implementa la función de barrera logarítmica.
  2. Desarrolla el algoritmo de punto interior.
  3. Visualiza la región factible, las curvas de nivel de la función objetivo y la trayectoria de optimización.
  4. Compara la solución con el método del Lagrangiano Aumentado.
import numpy as np
import matplotlib.pyplot as plt
from matplotlib.patches import Polygon

def funcion_objetivo(x, y):
    """Función objetivo: (x-4)² + (y-4)²"""
    return (x - 4)**2 + (y - 4)**2

def restriccion(x, y):
    """Restricción: x + y <= 5"""
    return x + y - 5  # <= 0

def barrera_logaritmica(x, y, t=1.0):
    """Función de barrera logarítmica"""
    # Restricciones: x + y <= 5, x >= 0, y >= 0
    if x <= 0 or y <= 0 or (x + y) >= 5 + 1e-6:  # Evitar log(0) o log(negativo)
        return np.inf
    
    # Término de barrera para x + y <= 5
    barrera = -np.log(5 - x - y)
    
    # Términos de barrera para x >= 0, y >= 0
    barrera -= np.log(x)
    barrera -= np.log(y)
    
    return barrera

def funcion_aumentada(x, y, t=1.0):
    """Función objetivo aumentada con barrera logarítmica"""
    return t * funcion_objetivo(x, y) + barrera_logaritmica(x, y, t)

def gradiente_funcion_aumentada(x, y, t=1.0):
    """Gradiente de la función aumentada"""
    # Gradiente de la función objetivo
    df_dx = 2 * (x - 4)
    df_dy = 2 * (y - 4)
    
    # Gradiente de la barrera logarítmica
    if x <= 0 or y <= 0 or (x + y) >= 5:
        return np.array([np.inf, np.inf])
    
    # Términos de la barrera
    dbarrier_dx = 1/(5 - x - y) - 1/x
    dbarrier_dy = 1/(5 - x - y) - 1/y
    
    # Gradiente total
    grad_x = t * df_dx + dbarrier_dx
    grad_y = t * df_dy + dbarrier_dy
    
    return np.array([grad_x, grad_y])

def metodo_punto_interior(x0, y0, t0=1.0, mu=10, max_iter_outer=10, max_iter_inner=1000, tol=1e-6):
    """Método de punto interior con barrera logarítmica"""
    x, y = x0, y0
    t = t0
    
    history = [(x, y)]
    
    for k in range(max_iter_outer):
        print(f"\nIteración exterior {k+1}, t = {t:.4f}")
        
        # Minimizar la función aumentada con el valor actual de t
        x_prev, y_prev = x, y
        
        for i in range(max_iter_inner):
            # Calcular gradiente
            grad = gradiente_funcion_aumentada(x, y, t)
            
            if np.any(np.isinf(grad)):
                print("Advertencia: Punto fuera del dominio de la barrera")
                break
            
            # Tasa de aprendizaje (puede ser adaptativa)
            alpha = 0.1 / np.sqrt(k+1)
            
            # Actualizar parámetros
            x_new = x - alpha * grad[0]
            y_new = y - alpha * grad[1]
            
            # Proyectar al conjunto factible si es necesario
            if x_new <= 0 or y_new <= 0 or (x_new + y_new) >= 5:
                # Si nos salimos del dominio, reducimos el paso
                alpha *= 0.5
                x_new = x - alpha * grad[0]
                y_new = y - alpha * grad[1]
            
            # Verificar convergencia
            if np.sqrt((x_new - x)**2 + (y_new - y)**2) < tol:
                print(f"  Convergencia alcanzada en la iteración {i+1}")
                break
                
            x, y = x_new, y_new
            
            # Guardar solo algunas iteraciones para visualización
            if i % 10 == 0:
                history.append((x, y))
        else:
            print(f"  Se alcanzó el número máximo de iteraciones internas")
        
        # Verificar convergencia del método exterior
        if np.sqrt((x - x_prev)**2 + (y - y_prev)**2) < tol and k > 0:
            print("Convergencia del método de punto interior alcanzada")
            break
        
        # Actualizar parámetro de barrera
        t = mu * t
    
    return np.array(history)

# Punto inicial factible
x0, y0 = 1.0, 1.0

# Ejecutar método de punto interior

Resumen

  • La optimización matemática es fundamental en IA y machine learning para entrenar modelos y ajustar parámetros.
  • Los problemas de optimización pueden clasificarse en lineales, no lineales, convexos, enteros, etc., según las propiedades de la función objetivo y las restricciones.
  • El gradiente descendente es un algoritmo iterativo que se mueve en la dirección opuesta al gradiente para encontrar mínimos locales.
  • El método de Newton utiliza información de segundo orden (Hessiana) para una convergencia más rápida, pero puede ser costoso computacionalmente.
  • Para problemas con restricciones, métodos como el Lagrangiano Aumentado y Barrera Logarítmica permiten transformar el problema en uno sin restricciones.
  • La optimización convexa es particularmente importante porque cualquier mínimo local es también global, y existen algoritmos eficientes para resolver estos problemas.

1. Introducción a las Excepciones

Una excepción es un evento que ocurre durante la ejecución de un programa que interrumpe el flujo normal de ejecución. En Python, cuando ocurre un error, se genera una excepción que, si no se maneja, detiene el programa y muestra un mensaje de error.

Ejemplo de error sin manejo

# Intentar dividir por cero genera una excepción
resultado = 10 / 0  # ZeroDivisionError: division by zero

1.1 Tipos de Errores Comunes

2. Manejo de Excepciones con try-except

2.1 Sintaxis Básica

El bloque try-except permite capturar y manejar excepciones.

Ejemplo básico de try-except

try:
    # Código que podría generar una excepción
    resultado = 10 / 0
except ZeroDivisionError:
    # Se ejecuta si ocurre ZeroDivisionError
    print("¡Error: No se puede dividir por cero!")
else:
    # Se ejecuta si no hay excepciones
    print(f"El resultado es {resultado}")
finally:
    # Siempre se ejecuta, haya o no excepciones
    print("Fin del bloque try-except")

2.2 Múltiples bloques except

Puedes capturar diferentes tipos de excepciones con múltiples bloques except.

Manejo de múltiples excepciones

try:
    # Código que podría generar diferentes excepciones
    indice = int(input("Ingrese un índice: "))
    lista = [10, 20, 30]
    print(f"El valor es: {lista[indice]}")
except ValueError:
    print("Error: Debe ingresar un número entero")
except IndexError:
    print(f"Error: El índice debe estar entre 0 y {len(lista)-1}")
except Exception as e:
    print(f"Ocurrió un error inesperado: {e}")

2.3 Captura de múltiples excepciones

Puedes capturar múltiples excepciones en una sola línea.

Captura de múltiples excepciones

try:
    # Código que podría generar excepciones
    valor = int(input("Ingrese un número: "))
    resultado = 100 / valor
except (ValueError, ZeroDivisionError) as e:
    print(f"Error: {e}")
    if isinstance(e, ValueError):
        print("Debe ingresar un número entero")
    else:
        print("No se puede dividir por cero")

Advertencia

Evita capturar la excepción genérica Exception sin una buena razón, ya que puede ocultar errores de programación. Es mejor capturar excepciones específicas.

3. La cláusula else y finally

3.1 La cláusula else

El bloque else se ejecuta solo si no se produjeron excepciones en el bloque try.

Uso de else con try-except

try:
    numero = int(input("Ingrese un número: "))
    resultado = 100 / numero
except (ValueError, ZeroDivisionError) as e:
    print(f"Error: {e}")
else:
    print(f"El resultado es: {resultado}")
    # Este bloque solo se ejecuta si no hubo excepciones
    print("Operación completada con éxito")

3.2 La cláusula finally

El bloque finally se ejecuta siempre, haya o no excepciones. Es útil para tareas de limpieza.

Uso de finally para limpieza

archivo = None
try:
    archivo = open("datos.txt", "r")
    contenido = archivo.read()
    # Procesar el contenido
    print(contenido)
except FileNotFoundError:
    print("Error: El archivo no existe")
finally:
    # Asegurarse de que el archivo se cierre
    if archivo is not None:
        archivo.close()
        print("Archivo cerrado correctamente")

Mejor práctica

Usa el contexto with para manejar recursos como archivos, ya que se encarga automáticamente de cerrarlos, incluso si ocurre una excepción.

# Forma recomendada para trabajar con archivos
try:
    with open("datos.txt", "r") as archivo:
        contenido = archivo.read()
        print(contenido)
except FileNotFoundError:
    print("Error: El archivo no existe")

4. Excepciones Personalizadas

Puedes crear tus propias excepciones heredando de la clase Exception o de alguna de sus subclases.

Creación de una excepción personalizada

class ValorDemasiadoPequenoError(ValueError):
    """Excepción lanzada cuando un valor es demasiado pequeño."""
    def __init__(self, valor, minimo):
        self.valor = valor
        self.minimo = minimo
        mensaje = f"El valor {valor} es menor que el mínimo permitido ({minimo})"
        super().__init__(mensaje)

def validar_edad(edad):
    if edad < 18:
        raise ValorDemasiadoPequenoError(edad, 18)
    return True

try:
    edad = int(input("Ingrese su edad: "))
    validar_edad(edad)
    print("¡Acceso concedido!")
except ValorDemasiadoPequenoError as e:
    print(f"Error: {e}")
    print(f"Valor proporcionado: {e.valor}, Mínimo requerido: {e.minimo}")
except ValueError:
    print("Error: Debe ingresar un número entero válido")

4.1 Buenas prácticas para excepciones personalizadas

  • Hereda de Exception o de una de sus subclases existentes.
  • Usa nombres que terminen en "Error" para mantener consistencia.
  • Proporciona mensajes de error descriptivos.
  • Documenta las excepciones personalizadas con docstrings.
  • Agrega atributos adicionales si es necesario para proporcionar más contexto.

5. Buenas Prácticas en el Manejo de Excepciones

5.1 Qué hacer y qué no hacer

Hacer

  • Captura excepciones específicas en lugar de la genérica Exception.
  • Proporciona mensajes de error útiles y descriptivos.
  • Registra las excepciones para depuración.
  • Limpia los recursos en bloques finally o usa el contexto with.
  • Documenta las excepciones que puede lanzar una función.

No hacer

  • No uses excepciones para controlar el flujo normal del programa.
  • No captures excepciones sin manejarlas adecuadamente.
  • No expongas información sensible en los mensajes de error.
  • No uses excepciones para validaciones que pueden hacerse con condicionales.

5.2 Registro de excepciones

Es una buena práctica registrar las excepciones para facilitar la depuración.

Registro de excepciones con el módulo logging

import logging

# Configuración básica del registro
logging.basicConfig(
    filename='app.log',
    level=logging.ERROR,
    format='%(asctime)s - %(name)s - %(levelname)s - %(message)s'
)

def dividir(a, b):
    try:
        return a / b
    except ZeroDivisionError as e:
        logging.error(f"Intento de división por cero: {a}/{b}", exc_info=True)
        raise  # Relanza la excepción después de registrarla
    except Exception as e:
        logging.error(f"Error inesperado: {e}", exc_info=True)
        raise

# Ejemplo de uso
try:
    resultado = dividir(10, 0)
except Exception as e:
    print("Ocurrió un error. Por favor, revise el archivo de registro.")

Ejercicio 1: Calculadora Segura

Crea una calculadora que maneje adecuadamente las excepciones al realizar operaciones matemáticas.

class OperacionInvalidaError(Exception):
    """Excepción para operaciones inválidas."""
    pass

def calcular():
    """Calculadora que maneja excepciones."""
    print("Calculadora Segura")
    print("Operaciones disponibles: +, -, *, /, ** (potencia)")
    
    while True:
        try:
            # Obtener entrada del usuario
            entrada = input("\nIngrese una operación (ej: 5 + 3) o 'salir' para terminar: ").strip()
            
            if entrada.lower() == 'salir':
                print("¡Hasta luego!")
                break
                
            # Dividir la entrada en componentes
            partes = entrada.split()
            
            if len(partes) != 3:
                raise ValueError("Formato incorrecto. Use: número operador número")
                
            num1_str, operador, num2_str = partes
            
            # Validar números
            try:
                num1 = float(num1_str)
                num2 = float(num2_str)
            except ValueError:
                raise ValueError("Ambos operandos deben ser números")
            
            # Realizar operación
            if operador == '+':
                resultado = num1 + num2
            elif operador == '-':
                resultado = num1 - num2
            elif operador == '*':
                resultado = num1 * num2
            elif operador == '/':
                if num2 == 0:
                    raise ZeroDivisionError("No se puede dividir por cero")
                resultado = num1 / num2
            elif operador == '**':
                resultado = num1 ** num2
            else:
                raise OperacionInvalidaError(f"Operador no soportado: {operador}")
            
            # Mostrar resultado
            print(f"Resultado: {resultado}")
            
        except ZeroDivisionError as e:
            print(f"Error: {e}")
        except ValueError as e:
            print(f"Error de valor: {e}")
        except OperacionInvalidaError as e:
            print(f"Error de operación: {e}")
        except Exception as e:
            print(f"Error inesperado: {e}")
        finally:
            print("-" * 30)

if __name__ == "__main__":
    calcular()

Ejercicio 2: Gestor de Contactos con Manejo de Errores

Mejora el gestor de contactos de sesiones anteriores agregando un manejo robusto de excepciones.

import json
import os
from datetime import datetime

class ContactoExistenteError(Exception):
    """Excepción para contactos que ya existen."""
    pass

class ContactoNoEncontradoError(Exception):
    """Excepción para contactos que no existen."""
    pass

class FormatoInvalidoError(Exception):
    """Excepción para formatos de entrada inválidos."""
    pass

class GestorContactos:
    """Clase para gestionar contactos con manejo de excepciones."""
    
    def __init__(self, archivo_contactos='contactos.json'):
        """Inicializa el gestor con el archivo de contactos."""
        self.archivo_contactos = archivo_contactos
        self.contactos = self._cargar_contactos()
    
    def _cargar_contactos(self):
        """Carga los contactos desde el archivo JSON."""
        try:
            if os.path.exists(self.archivo_contactos):
                with open(self.archivo_contactos, 'r', encoding='utf-8') as f:
                    return json.load(f)
            return {}
        except (json.JSONDecodeError, IOError) as e:
            print(f"Error al cargar los contactos: {e}")
            return {}
    
    def _guardar_contactos(self):
        """Guarda los contactos en el archivo JSON."""
        try:
            with open(self.archivo_contactos, 'w', encoding='utf-8') as f:
                json.dump(self.contactos, f, indent=4, ensure_ascii=False)
            return True
        except IOError as e:
            print(f"Error al guardar los contactos: {e}")
            return False
    
    def validar_email(self, email):
        """Valida el formato de un correo electrónico."""
        if not isinstance(email, str) or '@' not in email or '.' not in email:
            raise FormatoInvalidoError("Formato de correo electrónico inválido")
        return email.lower()
    
    def validar_telefono(self, telefono):
        """Valida el formato de un número de teléfono."""
        if not isinstance(telefono, str) or not telefono.isdigit() or len(telefono) < 8:
            raise FormatoInvalidoError("El teléfono debe contener solo números y tener al menos 8 dígitos")
        return telefono
    
    def agregar_contacto(self, nombre, email, telefono):
        """Agrega un nuevo contacto."""
        try:
            # Validar entradas
            if not nombre or not isinstance(nombre, str):
                raise FormatoInvalidoError("El nombre es obligatorio y debe ser texto")
            
            email = self.validar_email(email)
            telefono = self.validar_telefono(telefono)
            
            # Verificar si el contacto ya existe
            if email in self.contactos:
                raise ContactoExistenteError(f"Ya existe un contacto con el email: {email}")
            
            # Agregar el contacto
            self.contactos[email] = {
                'nombre': nombre,
                'telefono': telefono,
                'fecha_creacion': datetime.now().strftime("%Y-%m-%d %H:%M:%S"),
                'fecha_actualizacion': datetime.now().strftime("%Y-%m-%d %H:%M:%S")
            }
            
            # Guardar cambios
            if self._guardar_contactos():
                print(f"Contacto '{nombre}' agregado correctamente.")
                return True
            return False
            
        except (FormatoInvalidoError, ContactoExistenteError) as e:
            print(f"Error: {e}")
            return False
        except Exception as e:
            print(f"Error inesperado al agregar contacto: {e}")
            return False
    
    def buscar_contacto(self, email):
        """Busca un contacto por email."""
        try:
            email = self.validar_email(email)
            if email not in self.contactos:
                raise ContactoNoEncontradoError(f"No se encontró ningún contacto con el email: {email}")
            return self.contactos[email]
        except (FormatoInvalidoError, ContactoNoEncontradoError) as e:
            print(f"Error: {e}")
            return None
        except Exception as e:
            print(f"Error inesperado al buscar contacto: {e}")
            return None
    
    def listar_contactos(self):
        """Lista todos los contactos."""
        if not self.contactos:
            print("No hay contactos guardados.")
            return
        
        print("\n--- Lista de Contactos ---")
        for email, datos in self.contactos.items():
            print(f"\nEmail: {email}")
            for clave, valor in datos.items():
                if clave != 'email':  # Ya mostramos el email como clave
                    print(f"{clave.capitalize()}: {valor}")
    
    def eliminar_contacto(self, email):
        """Elimina un contacto por email."""
        try:
            email = self.validar_email(email)
            if email not in self.contactos:
                raise ContactoNoEncontradoError(f"No se encontró ningún contacto con el email: {email}")
            
            nombre = self.contactos[email]['nombre']
            del self.contactos[email]
            
            if self._guardar_contactos():
                print(f"Contacto '{nombre}' eliminado correctamente.")
                return True
            return False
            
        except (FormatoInvalidoError, ContactoNoEncontradoError) as e:
            print(f"Error: {e}")
            return False
        except Exception as e:
            print(f"Error inesperado al eliminar contacto: {e}")
            return False

def mostrar_menu():
    """Muestra el menú de opciones."""
    print("\n--- Gestor de Contactos ---")
    print("1. Agregar contacto")
    print("2. Buscar contacto")
    print("3. Listar contactos")
    print("4. Eliminar contacto")
    print("5. Salir")
    return input("Seleccione una opción: ")

def main():
    """Función principal del programa."""
    gestor = GestorContactos()
    
    while True:
        try:
            opcion = mostrar_menu()
            
            if opcion == '1':
                print("\n--- Agregar Nuevo Contacto ---")
                nombre = input("Nombre: ").strip()
                email = input("Email: ").strip()
                telefono = input("Teléfono: ").strip()
                gestor.agregar_contacto(nombre, email, telefono)
                
            elif opcion == '2':
                print("\n--- Buscar Contacto ---")
                email = input("Ingrese el email del contacto: ").strip()
                contacto = gestor.buscar_contacto(email)
                if contacto:
                    print("\nContacto encontrado:")
                    for clave, valor in contacto.items():
                        print(f"{clave.capitalize()}: {valor}")
                
            elif opcion == '3':
                gestor.listar_contactos()
                
            elif opcion == '4':
                print("\n--- Eliminar Contacto ---")
                email = input("Ingrese el email del contacto a eliminar: ").strip()
                gestor.eliminar_contacto(email)
                
            elif opcion == '5':
                print("¡Hasta luego!")
                break
                
            else:
                print("\nOpción no válida. Por favor, intente de nuevo.")
                
        except KeyboardInterrupt:
            print("\n\nOperación cancelada por el usuario.")
            break
        except Exception as e:
            print(f"\nError inesperado: {e}")

if __name__ == "__main__":
    main()

Ejercicio 3: Validador de Contraseñas

Crea una función que valide contraseñas según ciertos criterios y lance excepciones personalizadas.

class LongitudInvalidaError(Exception):
    """Excepción para contraseñas con longitud inválida."""
    pass

class FaltaMayusculaError(Exception):
    """Excepción para contraseñas sin mayúsculas."""
    pass

class FaltaMinusculaError(Exception):
    """Excepción para contraseñas sin minúsculas."""
    pass

class FaltaDigitoError(Exception):
    """Excepción para contraseñas sin dígitos."""
    pass

class FaltaCaracterEspecialError(Exception):
    """Excepción para contraseñas sin caracteres especiales."""
    pass

def validar_contrasena(contrasena, longitud_minima=8, caracteres_especiales="!@#$%^&*()_+-=[]{}|;:,.<>?"):
    """
    Valida una contraseña según varios criterios.
    
    Args:
        contrasena (str): La contraseña a validar.
        longitud_minima (int): Longitud mínima requerida.
        caracteres_especiales (str): Caracteres especiales permitidos.
        
    Returns:
        bool: True si la contraseña es válida.
        
    Raises:
        LongitudInvalidaError: Si la longitud es menor a la requerida.
        FaltaMayusculaError: Si no hay letras mayúsculas.
        FaltaMinusculaError: Si no hay letras minúsculas.
        FaltaDigitoError: Si no hay dígitos.
        FaltaCaracterEspecialError: Si no hay caracteres especiales.
    """
    # Validar tipo de entrada
    if not isinstance(contrasena, str):
        raise TypeError("La contraseña debe ser una cadena de texto")
    
    # Validar longitud
    if len(contrasena) < longitud_minima:
        raise LongitudInvalidaError(
            f"La contraseña debe tener al menos {longitud_minima} caracteres"
        )
    
    # Verificar mayúsculas
    if not any(c.isupper() for c in contrasena):
        raise FaltaMayusculaError(
            "La contraseña debe contener al menos una letra mayúscula"
        )
    
    # Verificar minúsculas
    if not any(c.islower() for c in contrasena):
        raise FaltaMinusculaError(
            "La contraseña debe contener al menos una letra minúscula"
        )
    
    # Verificar dígitos
    if not any(c.isdigit() for c in contrasena):
        raise FaltaDigitoError(
            "La contraseña debe contener al menos un dígito"
        )
    
    # Verificar caracteres especiales
    if not any(c in caracteres_especiales for c in contrasena):
        raise FaltaCaracterEspecialError(
            f"La contraseña debe contener al menos un carácter especial: {caracteres_especiales}"
        )
    
    return True

def probar_contrasena():
    """Función para probar el validador de contraseñas."""
    print("Validador de Contraseñas")
    print("La contraseña debe cumplir con los siguientes requisitos:")
    print(f"- Mínimo 8 caracteres")
    print("- Al menos una letra mayúscula")
    print("- Al menos una letra minúscula")
    print("- Al menos un dígito")
    print("- Al menos un carácter especial: !@#$%^&*()_+-=[]{}|;:,.<>?")
    
    while True:
        try:
            contrasena = input("\nIngrese una contraseña (o 'salir' para terminar): ")
            
            if contrasena.lower() == 'salir':
                print("¡Hasta luego!")
                break
                
            if validar_contrasena(contrasena):
                print("¡Contraseña válida!")
                break
                
        except LongitudInvalidaError as e:
            print(f"Error de longitud: {e}")
        except FaltaMayusculaError as e:
            print(f"Error: {e}")
        except FaltaMinusculaError as e:
            print(f"Error: {e}")
        except FaltaDigitoError as e:
            print(f"Error: {e}")
        except FaltaCaracterEspecialError as e:
            print(f"Error: {e}")
        except Exception as e:
            print(f"Error inesperado: {e}")
        finally:
            print("-" * 50)

if __name__ == "__main__":
    probar_contrasena()

Introducción a la Selección de Características

La selección de características es un proceso fundamental en el aprendizaje automático que consiste en elegir un subconjunto de características relevantes para la construcción de modelos predictivos.

¿Por qué es importante?

  • Mejora el rendimiento del modelo
  • Reduce el sobreajuste (overfitting)
  • Disminuye el tiempo de entrenamiento
  • Facilita la interpretación del modelo

Importancia de las características (visualización interactiva)

Métodos de Selección de Características

1. Métodos de Filtrado

Selección basada en estadísticas sin considerar el modelo.

  • Correlación de Pearson
  • Prueba de Chi-cuadrado
  • Análisis de varianza (ANOVA)
  • Información mutua

2. Métodos de Envoltura (Wrapper)

Utilizan el modelo de aprendizaje automático para evaluar subconjuntos de características.

  • Selección hacia adelante (Forward Selection)
  • Eliminación hacia atrás (Backward Elimination)
  • Búsqueda exhaustiva

3. Métodos Integrados (Embedded)

La selección de características se realiza durante el proceso de entrenamiento.

  • Regularización L1 (Lasso)
  • Árboles de decisión y Random Forest
  • XGBoost, LightGBM

Implementación en Python

1. Usando SelectKBest (Método de Filtrado)

from sklearn.feature_selection import SelectKBest, f_classif
from sklearn.datasets import load_iris

# Cargar datos
X, y = load_iris(return_X_y=True)

# Seleccionar las 2 mejores características
selector = SelectKBest(score_func=f_classif, k=2)
X_new = selector.fit_transform(X, y)

# Ver características seleccionadas
print("Características seleccionadas:", selector.get_support(indices=True))

2. Usando SelectFromModel (Método Integrado)

from sklearn.ensemble import RandomForestClassifier
from sklearn.feature_selection import SelectFromModel

# Entrenar modelo
clf = RandomForestClassifier(n_estimators=100, random_state=42)
clf = clf.fit(X, y)

# Seleccionar características importantes
model = SelectFromModel(clf, prefit=True)
X_new = model.transform(X)

# Ver importancia de características
print("Importancia de características:", clf.feature_importances_)

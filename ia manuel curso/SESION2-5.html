Contenido Teórico

¿Qué es el cálculo diferencial?

El cálculo diferencial es una rama de las matemáticas que estudia cómo cambian las funciones cuando sus variables cambian. Su concepto principal es la derivada.

La derivada de una función en un punto representa la pendiente de la recta tangente a la curva en ese punto, lo que equivale a la tasa de cambio instantánea de la función.

\[ f'(x) = \lim_{h \to 0} \frac{f(x + h) - f(x)}{h} \]

Dato clave: En aprendizaje automático, las derivadas son fundamentales para el algoritmo de descenso de gradiente, que se utiliza para optimizar modelos.

Reglas básicas de derivación

1. Derivada de una constante

\[ \frac{d}{dx}(c) = 0 \]

2. Regla de la potencia

\[ \frac{d}{dx}(x^n) = nx^{n-1} \]

3. Regla de la suma/resta

\[ \frac{d}{dx}[f(x) \pm g(x)] = f'(x) \pm g'(x) \]

4. Regla del producto

\[ \frac{d}{dx}[f(x) \cdot g(x)] = f'(x) \cdot g(x) + f(x) \cdot g'(x) \]

5. Regla del cociente

\[ \frac{d}{dx}\left(\frac{f(x)}{g(x)}\right) = \frac{f'(x)g(x) - f(x)g'(x)}{[g(x)]^2} \]

Derivadas de funciones comunes

Función Derivada
\( \sin(x) \) \( \cos(x) \)
\( \cos(x) \) \( -\sin(x) \)
\( e^x \) \( e^x \)
\( \ln(x) \) \( \frac{1}{x} \)

Actividad Práctica

1. Cálculo de derivadas con SymPy

Vamos a calcular derivadas usando la biblioteca SymPy en Python, que nos permite hacer cálculo simbólico.

Código en Python:

from sympy import symbols, diff, exp, sin, cos, ln, simplify

# Definir la variable simbólica
x = symbols('x')

# Ejemplo 1: Derivada de un polinomio
f1 = x**3 - 2*x**2 + 5*x - 3
df1 = diff(f1, x)
print("1. Derivada de", f1, ":")
print("   ", df1)

# Ejemplo 2: Derivada de una función trigonométrica
f2 = sin(x) * cos(x)
df2 = diff(f2, x)
print("\n2. Derivada de", f2, ":")
print("   ", simplify(df2))  # simplify simplifica la expresión

# Ejemplo 3: Derivada de una función exponencial
f3 = exp(2*x) + ln(x)
df3 = diff(f3, x)
print("\n3. Derivada de", f3, ":")
print("   ", df3)
Explicación del código

Este código utiliza la biblioteca SymPy para cálculo simbólico. La función diff() calcula la derivada de una expresión con respecto a la variable especificada.

El método simplify() se utiliza para simplificar expresiones matemáticas cuando sea posible.

2. Visualización de la derivada como pendiente

Vamos a visualizar cómo la derivada representa la pendiente de la recta tangente en un punto.

import numpy as np
import matplotlib.pyplot as plt

def f(x):
    return x**2  # Función cuadrática

def df(x):
    return 2*x  # Derivada de x^2 es 2x

# Punto donde calcularemos la recta tangente
x0 = 1.5
y0 = f(x0)
m = df(x0)  # Pendiente de la recta tangente

# Crear datos para la gráfica
x = np.linspace(-2, 3, 400)
y = f(x)

# Ecuación de la recta tangente: y = m(x - x0) + y0
tangent_line = m * (x - x0) + y0

# Crear la gráfica
plt.figure(figsize=(10, 6))
plt.plot(x, y, label='f(x) = x²', linewidth=2)
plt.plot(x, tangent_line, 'r--', label=f'Recta tangente en x={x0}')
plt.scatter([x0], [y0], color='red')  # Punto de tangencia

# Configuración de la gráfica
plt.axhline(0, color='black', linewidth=0.5)
plt.axvline(0, color='black', linewidth=0.5)
plt.grid(True, linestyle='--', alpha=0.7)
plt.title('Interpretación geométrica de la derivada')
plt.xlabel('x')
plt.ylabel('y')
plt.legend()
plt.axis('equal')
plt.show()

En esta visualización:

  • La curva azul representa la función cuadrática f(x) = x².
  • La línea roja discontinua es la recta tangente a la curva en el punto x=1.5.
  • La pendiente de esta recta tangente es igual al valor de la derivada en ese punto (que es 3 cuando x=1.5).
  • Puedes cambiar el valor de x0 para ver cómo cambia la recta tangente en diferentes puntos de la curva.

3. Aplicación: Encontrar mínimos de una función

Un uso común de las derivadas es encontrar mínimos (o máximos) de funciones, lo cual es fundamental en algoritmos de optimización como el descenso de gradiente.

from scipy.optimize import minimize_scalar

def f(x):
    return x**4 - 4*x**2 + 2*x + 5

# Encontrar el mínimo de la función
result = minimize_scalar(f)
min_x = result.x
min_y = result.fun

print(f"El mínimo de la función está en x = {min_x:.4f}, f(x) = {min_y:.4f}")

# Graficar la función y el mínimo encontrado
x = np.linspace(-2, 2, 400)
y = f(x)

plt.figure(figsize=(10, 6))
plt.plot(x, y, label='f(x) = x⁴ - 4x² + 2x + 5')
plt.scatter([min_x], [min_y], color='red', label=f'Mínimo en ({min_x:.2f}, {min_y:.2f})')
plt.axhline(0, color='black', linewidth=0.5)
plt.axvline(0, color='black', linewidth=0.5)
plt.grid(True, linestyle='--', alpha=0.7)
plt.title('Encontrar el mínimo de una función')
plt.xlabel('x')
plt.ylabel('y')
plt.legend()
plt.show()
Relación con el aprendizaje automático

En aprendizaje automático, el proceso de entrenamiento de un modelo a menudo implica minimizar una función de pérdida. El algoritmo de descenso de gradiente, que utiliza derivadas, es fundamental para este propósito.

La idea es similar al ejemplo mostrado, pero en múltiples dimensiones, donde la derivada se convierte en un gradiente (vector de derivadas parciales).

Resumen

  • La derivada mide la tasa de cambio instantánea de una función.
  • Las reglas básicas de derivación nos permiten encontrar derivadas de funciones complejas.
  • Geométricamente, la derivada en un punto es la pendiente de la recta tangente a la curva en ese punto.
  • En Python, podemos usar SymPy para cálculo simbólico de derivadas.
  • Las derivadas son fundamentales en algoritmos de optimización como el descenso de gradiente, ampliamente utilizado en aprendizaje automático.

Conceptos Básicos de Probabilidad

La probabilidad es una medida de la posibilidad de que ocurra un evento. Se expresa como un número entre 0 (imposible) y 1 (seguro).

\[ P(A) = \frac{\text{Número de resultados favorables}}{\text{Número total de resultados posibles}} \]

Ejemplo:

Al lanzar un dado justo de 6 caras, la probabilidad de obtener un 3 es:

\[ P(3) = \frac{1}{6} \approx 0.1667 \text{ o } 16.67\% \]

En IA: La probabilidad es fundamental para algoritmos como Naive Bayes, redes bayesianas y modelos generativos.

Reglas de Probabilidad

1. Regla de la Suma (Eventos Mutuamente Excluyentes)

\[ P(A \cup B) = P(A) + P(B) - P(A \cap B) \]

2. Regla del Producto (Eventos Independientes)

\[ P(A \cap B) = P(A) \times P(B) \quad \text{(si A y B son independientes)} \]

3. Probabilidad Condicional

\[ P(A|B) = \frac{P(A \cap B)}{P(B)} \quad \text{si } P(B) > 0 \]

4. Teorema de Bayes

\[ P(A|B) = \frac{P(B|A) \times P(A)}{P(B)} \]

Variables Aleatorias

Una variable aleatoria es una variable cuyos valores posibles son resultados numéricos de un fenómeno aleatorio.

Tipos de variables aleatorias:

  • Discretas: Toman valores contables (ej: número de caras en lanzamientos de moneda)
  • Continuas: Toman valores en un intervalo (ej: altura, peso, tiempo)

Valor Esperado (Esperanza Matemática)

\[ E[X] = \sum_{i} x_i \cdot P(x_i) \quad \text{(discreto)} \] \[ E[X] = \int_{-\infty}^{\infty} x \cdot f(x) \, dx \quad \text{(continuo)} \]

Teorema del Límite Central

Cuando el tamaño de la muestra es suficientemente grande, la distribución de la media muestral se aproxima a una distribución normal, independientemente de la forma de la distribución de la población.

\[ \bar{X} \sim N\left(\mu, \frac{\sigma^2}{n}\right) \]

Importante: Este teorema es fundamental en estadística inferencial y justifica el uso de la distribución normal en muchos contextos.

Actividad Práctica

1. Cálculo de Probabilidades con Python

Vamos a calcular probabilidades usando la biblioteca NumPy y SciPy.

Código en Python:

import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

# 1. Probabilidad de eventos simples
# Lanzamiento de una moneda justa
p_cara = 0.5
p_sello = 1 - p_cara
print(f"Probabilidad de cara: {p_cara}")
print(f"Probabilidad de sello: {p_sello}")

# 2. Probabilidad de eventos compuestos
# Probabilidad de obtener al menos un 6 en 4 lanzamientos de un dado
dado = np.arange(1, 7)  # Caras del dado
p_no_seis = (5/6) ** 4  # Probabilidad de no obtener 6 en 4 lanzamientos
p_al_menos_un_seis = 1 - p_no_seis
print(f"\nProbabilidad de al menos un 6 en 4 lanzamientos: {p_al_menos_un_seis:.4f}")

# 3. Distribución binomial
# Probabilidad de obtener exactamente 3 caras en 5 lanzamientos de moneda
n = 5  # Número de ensayos
k = 3  # Número de éxitos
p = 0.5  # Probabilidad de éxito
prob_binomial = stats.binom.pmf(k, n, p)
print(f"\nProbabilidad binomial (3 caras en 5 lanzamientos): {prob_binomial:.4f}")

# 4. Distribución normal
# Probabilidad de que una variable normal estándar sea menor que 1.5
prob_normal = stats.norm.cdf(1.5)  # Función de distribución acumulativa
print(f"\nP(Z < 1.5) = {prob_normal:.4f}")

# 5. Teorema de Bayes - Ejemplo de diagnóstico médico
# Sensibilidad (verdaderos positivos)
p_positivo_dado_enfermo = 0.99
# Especificidad (verdaderos negativos)
p_negativo_dado_sano = 0.98
# Prevalencia de la enfermedad
p_enfermo = 0.001

# Calcular P(Enfermo|Positivo) usando el Teorema de Bayes
p_positivo = (p_positivo_dado_enfermo * p_enfermo) + \
             ((1 - p_negativo_dado_sano) * (1 - p_enfermo))
             
p_enfermo_dado_positivo = (p_positivo_dado_enfermo * p_enfermo) / p_positivo

print(f"\nTeorema de Bayes - Ejemplo de diagnóstico médico:")
print(f"P(Enfermo|Positivo) = {p_enfermo_dado_positivo:.4f} o {p_enfermo_dado_positivo*100:.2f}%")
Explicación del código

Este código muestra varios conceptos de probabilidad:

  • Cálculo de probabilidades simples y compuestas
  • Uso de la distribución binomial para eventos discretos
  • Cálculo de probabilidades con la distribución normal
  • Aplicación del teorema de Bayes para actualizar creencias

2. Simulación del Teorema del Límite Central

Vamos a simular el teorema del límite central lanzando dados.

import numpy as np
import matplotlib.pyplot as plt

# Configuración
np.random.seed(42)  # Para reproducibilidad
n_simulaciones = 10000  # Número de muestras
n_lanzamientos = 30  # Tamaño de cada muestra

# Simular lanzamientos de dados (valores del 1 al 6)
lanzamientos = np.random.randint(1, 7, size=(n_simulaciones, n_lanzamientos))

# Calcular la media de cada muestra
medias_muestrales = np.mean(lanzamientos, axis=1)

# Parámetros teóricos
mu_teorico = 3.5  # Media de un dado justo
sigma_teorico = np.sqrt(35/12)  # Desviación estándar de un dado justo
sigma_medias_teorico = sigma_teorico / np.sqrt(n_lanzamientos)

# Crear la figura
plt.figure(figsize=(12, 6))

# Histograma de las medias muestrales
n, bins, patches = plt.hist(medias_muestrales, bins=30, density=True, 
                           alpha=0.7, color='skyblue', edgecolor='black')

# Dibujar la distribución normal teórica
from scipy.stats import norm
x = np.linspace(mu_teorico - 4*sigma_medias_teorico, 
                mu_teorico + 4*sigma_medias_teorico, 100)
plt.plot(x, norm.pdf(x, mu_teorico, sigma_medias_teorico), 
         'r-', lw=2, label='Distribución normal teórica')

# Línea vertical en la media
plt.axvline(mu_teorico, color='k', linestyle='dashed', linewidth=1)
plt.text(mu_teorico+0.1, 0.1, f'Media = {mu_teorico}', rotation=90, verticalalignment='bottom')

# Configuración de la gráfica
plt.title(f'Teorema del Límite Central\nDistribución de las medias de {n_lanzamientos} lanzamientos de dado')
plt.xlabel('Media muestral')
plt.ylabel('Densidad de probabilidad')
plt.legend()
plt.grid(True, linestyle='--', alpha=0.7)

# Mostrar estadísticas muestrales
media_muestral = np.mean(medias_muestrales)
desv_est_muestral = np.std(medias_muestrales, ddof=1)

plt.figtext(0.5, -0.1, 
            f'Media muestral = {media_muestral:.4f} | ' +
            f'Desv. estándar muestral = {desv_est_muestral:.4f}\n' +
            f'Media teórica = {mu_teorico} | ' +
            f'Error estándar teórico = {sigma_medias_teorico:.4f}',
            ha='center', fontsize=10, bbox={"facecolor":"white", "alpha":0.8, "pad":5})

plt.tight_layout()
plt.show()

Esta simulación demuestra el Teorema del Límite Central:

  • Generamos múltiples muestras de lanzamientos de dados.
  • Calculamos la media de cada muestra.
  • Observamos que la distribución de estas medias se aproxima a una distribución normal.
  • La media de la distribución de medias se acerca a la media poblacional (3.5 para un dado justo).
  • La desviación estándar de las medias (error estándar) disminuye a medida que aumenta el tamaño de la muestra.

3. Aplicación: Filtro de Spam con Naive Bayes

Vamos a implementar un clasificador de spam simple usando el teorema de Bayes.

import numpy as np
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.naive_bayes import MultinomialNB
from sklearn.metrics import accuracy_score, confusion_matrix
import seaborn as sns
import matplotlib.pyplot as plt

# Conjunto de datos de ejemplo (en la práctica, usaríamos un conjunto más grande)
correos = [
    "oferta especial solo hoy",
    "reunión importante mañana a las 3pm",
    "gana dinero rápido y fácil",
    "recordatorio: entrega del proyecto",
    "trabajo desde casa sin inversión",
    "revisión de código pendiente",
    "millonario en una semana",
    "actualización del sistema programada"
]

# Etiquetas: 1 para spam, 0 para no spam
y = np.array([1, 0, 1, 0, 1, 0, 1, 0])

# Convertir texto a vectores de conteo
vectorizer = CountVectorizer()
X = vectorizer.fit_transform(correos)

# Entrenar el clasificador Naive Bayes
modelo = MultinomialNB()
modelo.fit(X, y)

# Hacer predicciones
predicciones = modelo.predict(X)

# Evaluar el modelo
precision = accuracy_score(y, predicciones)
print(f"Precisión del modelo: {precision*100:.2f}%")

# Matriz de confusión
matriz_confusion = confusion_matrix(y, predicciones)
plt.figure(figsize=(6, 4))
sns.heatmap(matriz_confusion, annot=True, fmt='d', cmap='Blues', 
            xticklabels=['No Spam', 'Spam'],
            yticklabels=['No Spam', 'Spam'])
plt.xlabel('Predicción')
plt.ylabel('Real')
plt.title('Matriz de Confusión')
plt.show()

# Probar con nuevos correos
nuevos_correos = [
    "oferta de trabajo desde casa",
    "reunión de equipo mañana",
    "gane dinero fácil con criptomonedas"
]

X_nuevos = vectorizer.transform(nuevos_correos)
predicciones_nuevas = modelo.predict(X_nuevos)

print("\nPredicciones para nuevos correos:")
for correo, pred in zip(nuevos_correos, predicciones_nuevas):
    print(f"\nCorreo: {correo}")
    print(f"Predicción: {'Spam' if pred == 1 else 'No Spam'}")
    # Probabilidades para cada clase
    probas = modelo.predict_proba(vectorizer.transform([correo]))[0]
    print(f"Probabilidades: [No Spam: {probas[0]:.4f}, Spam: {probas[1]:.4f}]")
Explicación del clasificador

Este código implementa un clasificador Naive Bayes para detectar correos spam:

  • Usa el teorema de Bayes para calcular la probabilidad de que un correo sea spam.
  • La suposición "ingenua" es que las palabras son independientes entre sí.
  • El modelo aprende las probabilidades de cada palabra en correos spam y no spam.
  • Para nuevos correos, calcula la probabilidad de que sea spam basándose en las palabras que contiene.
  • En la práctica, se usaría un conjunto de datos mucho más grande para entrenar el modelo.

Resumen

  • La probabilidad mide la posibilidad de que ocurra un evento, con valores entre 0 (imposible) y 1 (seguro).
  • Las reglas de probabilidad (suma, producto, probabilidad condicional) son fundamentales para calcular probabilidades de eventos complejos.
  • El teorema de Bayes permite actualizar creencias a medida que obtenemos nueva evidencia.
  • El teorema del límite central explica por qué muchas mediciones siguen una distribución normal.
  • Los algoritmos de IA como Naive Bayes utilizan conceptos de probabilidad para hacer predicciones.

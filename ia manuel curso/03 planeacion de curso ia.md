# Plan de Curso: De Cero a Experto en Inteligencia Artificial

## Información General

- **Duración**: 100 horas (100 sesiones de 1 hora)
- **Nivel**: De principiante a experto
- **Requisitos previos**: Ninguno (se empieza desde cero)
- **Modalidad**: Teórico-práctica

## Objetivos del Curso

1. Adquirir bases sólidas en programación Python
2. Comprender los fundamentos matemáticos de la IA
3. Aprender sobre Machine Learning y Deep Learning
4. Desarrollar proyectos prácticos de IA
5. Implementar modelos de IA desde cero
6. Desplegar modelos en producción

## Hardware Requerido

- Computadora con al menos 8GB de RAM (16GB recomendado)
- Procesador de 4 núcleos o superior
- Tarjeta gráfica NVIDIA (recomendada para Deep Learning)
- 10GB de espacio libre en disco
- Conexión a Internet estable

## Software Requerido

1. **Sistema Operativo**: Windows 10/11, macOS o Linux
2. **Python 3.8+**
3. **Entornos de Desarrollo**:
   - Visual Studio Code
   - Jupyter Notebook
   - Google Colab (opcional)
4. **Bibliotecas Principales**:
   - NumPy, Pandas, Matplotlib
   - Scikit-learn
   - TensorFlow / PyTorch
   - OpenCV (procesamiento de imágenes)
   - NLTK (procesamiento de lenguaje natural)

## Estructura del Curso

### Módulo 1: Introducción a la Programación con Python (Sesiones 1-15)

- Instalación y configuración del entorno
- Fundamentos de programación
- Estructuras de datos
- Programación orientada a objetos
- Manejo de archivos y excepciones

### Módulo 2: Matemáticas para IA (Sesiones 16-30)

- Álgebra lineal
- Cálculo diferencial e integral
- Probabilidad y estadística
- Optimización

### Módulo 3: Fundamentos de Machine Learning (Sesiones 31-50)

- Introducción al Aprendizaje Automático
- Aprendizaje supervisado y no supervisado
- Regresión y clasificación
- Árboles de decisión y bosques aleatorios
- Redes neuronales básicas

### Módulo 4: Deep Learning (Sesiones 51-80)

- Redes Neuronales Artificiales
- Redes Neuronales Convolucionales (CNN)
- Redes Neuronales Recurrentes (RNN)
- Redes Generativas Adversarias (GANs)
- Procesamiento de Lenguaje Natural (NLP)

### Módulo 5: Proyectos Avanzados (Sesiones 81-100)

- Proyecto 1: Sistema de reconocimiento de imágenes
- Proyecto 2: Chatbot inteligente
- Proyecto 3: Sistema de recomendación
- Despliegue de modelos
- Ética en IA

## Evaluación

- Ejercicios prácticos semanales (40%)
- Proyectos por módulo (40%)
- Examen final (20%)

## Recursos Adicionales

- Libros recomendados
- Artículos de investigación
- Comunidades en línea
- Competencias de IA

1. Introducción a los Módulos en Python

Un módulo en Python es un archivo que contiene definiciones y sentencias de Python. Los módulos permiten organizar el código de manera lógica y reutilizable. Cualquier archivo con extensión .py es un módulo en Python.

Ventajas de usar módulos:

  • Reutilización de código: Permite usar el mismo código en diferentes programas.
  • Organización: Facilita la estructuración lógica del código.
  • Mantenimiento: Facilita la actualización y corrección de errores en un solo lugar.
  • Espacios de nombres: Evita colisiones de nombres entre diferentes partes de un programa.

1.1 Creación y uso de módulos

Para crear un módulo, simplemente guardamos nuestro código en un archivo con extensión .py.

Ejemplo: Creación de un módulo simple

Creemos un módulo llamado matematicas.py:

# matematicas.py
"""
Módulo de operaciones matemáticas básicas.
Incluye funciones para operaciones comunes como suma, resta, etc.
"""

def suma(a, b):
    """Devuelve la suma de dos números."""
    return a + b

def resta(a, b):
    """Devuelve la resta de dos números."""
    return a - b

def multiplicacion(a, b):
    """Devuelve el producto de dos números."""
    return a * b

def division(a, b):
    """
    Devuelve la división de dos números.
    Lanza una excepción si el divisor es cero.
    """
    if b == 0:
        raise ValueError("No se puede dividir por cero")
    return a / b

# Variable del módulo
PI = 3.14159265359

# Ejecución condicional para pruebas
if __name__ == "__main__":
    # Código de prueba
    print("Probando el módulo matemáticas:")
    print(f"Suma: 5 + 3 = {suma(5, 3)}")
    print(f"Resta: 5 - 3 = {resta(5, 3)}")
    print(f"Multiplicación: 5 * 3 = {multiplicacion(5, 3)}")
    try:
        print(f"División: 5 / 3 = {division(5, 3):.2f}")
        print(f"División por cero: {division(5, 0):.2f}")
    except ValueError as e:
        print(f"Error: {e}")

1.2 Importación de módulos

Existen varias formas de importar módulos en Python. Veamos las más comunes:

Ejemplo: Diferentes formas de importar módulos

# 1. Importar el módulo completo
import matematicas

# Usar funciones con notación de punto
resultado = matematicas.suma(10, 5)
print(f"10 + 5 = {resultado}")

# 2. Importar funciones específicas
from matematicas import resta, multiplicacion

# Usar las funciones importadas directamente
print(f"10 - 5 = {resta(10, 5)}")
print(f"10 * 5 = {multiplicacion(10, 5)}")

# 3. Importar con un alias
import matematicas as m

# Usar el alias
print(f"Valor de PI: {m.PI}")

# 4. Importar todas las funciones (no recomendado en producción)
# from matematicas import *

# 5. Importar con alias para funciones específicas
from matematicas import division as div

try:
    print(f"10 / 5 = {div(10, 5)}")
    print(f"10 / 0 = {div(10, 0)}")
except ValueError as e:
    print(f"Error: {e}")

# Verificar el nombre del módulo
print(f"El nombre del módulo es: {matematicas.__name__}")
print(f"La documentación del módulo: {matematicas.__doc__}")
print(f"Documentación de la función suma: {matematicas.suma.__doc__}")

Buenas prácticas al importar módulos

  • Evita usar from modulo import * ya que puede causar colisiones de nombres.
  • Usa alias cortos pero descriptivos (ej: import numpy as np).
  • Agrupa las importaciones en este orden:
    1. Módulos estándar de Python
    2. Módulos de terceros
    3. Módulos de la aplicación
  • Usa if __name__ == "__main__" para código que solo debe ejecutarse cuando el módulo se ejecuta directamente.

2. Paquetes en Python

Un paquete es una forma de organizar módulos relacionados en una jerarquía de directorios. Los paquetes permiten una estructura de espacio de nombres plana para evitar colisiones de nombres entre módulos.

2.1 Estructura de un paquete

Un paquete es un directorio que contiene un archivo especial llamado __init__.py (que puede estar vacío) y uno o más módulos.

mi_paquete/
├── __init__.py
├── modulo1.py
├── modulo2.py
└── subpaquete/
├── __init__.py
└── modulo3.py

Ejemplo: Creación de un paquete simple

1. Primero, crea la siguiente estructura de directorios:

mkdir -p mi_paquete/subpaquete
touch mi_paquete/__init__.py
touch mi_paquete/modulo1.py
touch mi_paquete/modulo2.py
touch mi_paquete/subpaquete/__init__.py
touch mi_paquete/subpaquete/modulo3.py

4. Instalación de Paquetes Externos

Python tiene un rico ecosistema de paquetes de terceros que puedes instalar usando pip, el administrador de paquetes de Python.

4.1 Uso básico de pip

Comandos básicos de pip

# Instalar un paquete
pip install nombre_del_paquete

# Instalar una versión específica
pip install nombre_del_paquete==1.2.3

# Actualizar un paquete
pip install --upgrade nombre_del_paquete

# Desinstalar un paquete
pip uninstall nombre_del_paquete

# Listar paquetes instalados
pip list

# Ver información de un paquete
pip show nombre_del_paquete

# Guardar dependencias en un archivo requirements.txt
pip freeze > requirements.txt

# Instalar dependencias desde requirements.txt
pip install -r requirements.txt

4.2 Entornos virtuales

Los entornos virtuales permiten aislar las dependencias de diferentes proyectos. Es una buena práctica usar un entorno virtual para cada proyecto.

Uso de entornos virtuales

# Crear un entorno virtual (en Windows)
python -m venv venv

# Activar el entorno virtual (Windows)
.\venv\Scripts\activate

# Crear un entorno virtual (en macOS/Linux)
python3 -m venv venv

# Activar el entorno virtual (macOS/Linux)
source venv/bin/activate

# Instalar paquetes en el entorno virtual
pip install requests pandas numpy

# Desactivar el entorno virtual
deactivate

Importancia de los entornos virtuales

Usar entornos virtuales es esencial para:

  • Evitar conflictos de versiones entre proyectos.
  • Mantener limpio el entorno global de Python.
  • Facilitar la replicación del entorno de desarrollo.
  • Gestionar dependencias de manera más efectiva.

"""
Funciones estadísticas básicas.
"""
from typing import List, Union, Dict, Any
import math

def media(numeros: List[float]) -> float:
    """Calcula la media aritmética de una lista de números."""
    if not numeros:
        raise ValueError("La lista de números no puede estar vacía")
    return sum(numeros) / len(numeros)

def mediana(numeros: List[float]) -> float:
    """Calcula la mediana de una lista de números."""
    if not numeros:
        raise ValueError("La lista de números no puede estar vacía")
        
    numeros_ordenados = sorted(numeros)
    n = len(numeros_ordenados)
    mitad = n // 2
    
    if n % 2 == 1:
        return numeros_ordenados[mitad]
    else:
        return (numeros_ordenados[mitad - 1] + numeros_ordenados[mitad]) / 2

def moda(numeros: List[float]) -> List[float]:
    """
    Calcula la moda de una lista de números.
    
    Returns:
        List[float]: Lista de modas (puede haber más de una)
    """
    if not numeros:
        raise ValueError("La lista de números no puede estar vacía")
        
    frecuencias: Dict[float, int] = {}
    
    for num in numeros:
        if num in frecuencias:
            frecuencias[num] += 1
        else:
            frecuencias[num] = 1
    
    max_frecuencia = max(frecuencias.values())
    modas = [num for num, freq in frecuencias.items() if freq == max_frecuencia]
    
    return modas

def desviacion_estandar(numeros: List[float], poblacional: bool = True) -> float:
    """
    Calcula la desviación estándar de una lista de números.
    
    Args:
        numeros: Lista de números
        poblacional: Si es True, calcula la desviación estándar poblacional.
                    Si es False, calcula la desviación estándar muestral.
    """
    if not numeros:
        raise ValueError("La lista de números no puede estar vacía")
        
    n = len(numeros)
    if n == 1:
        return 0.0
        
    m = media(numeros)
    suma_cuadrados = sum((x - m) ** 2 for x in numeros)
    
    # Usar n-1 para la desviación estándar muestral (corrección de Bessel)
    divisor = n if poblacional else (n - 1)
    
    return math.sqrt(suma_cuadrados / divisor)

__all__ = ['media', 'mediana', 'moda', 'desviacion_estandar']

"""
Funciones para manipulación de texto.
"""
import re
from typing import List, Dict, Tuple, Union, Optional

def invertir_texto(texto: str) -> str:
    """Invierte el orden de los caracteres en un texto."""
    return texto[::-1]

def contar_vocales(texto: str) -> Dict[str, int]:
    """
    Cuenta la cantidad de cada vocal en un texto (sin distinguir mayúsculas/minúsculas).
    
    Returns:
        Dict[str, int]: Diccionario con el conteo de cada vocal
    """
    texto = texto.lower()
    vocales = {'a': 0, 'e': 0, 'i': 0, 'o': 0, 'u': 0}
    
    for letra in texto:
        if letra in vocales:
            vocales[letra] += 1
    
    return vocales

def es_palindromo(texto: str, ignorar_espacios: bool = True, 
                 ignorar_acentos: bool = True, 
                 ignorar_mayusculas: bool = True) -> bool:
    """
    Determina si un texto es un palíndromo.
    
    Args:
        texto: Texto a evaluar
        ignorar_espacios: Si es True, ignora los espacios en blanco
        ignorar_acentos: Si es True, ignora los acentos
        ignorar_mayusculas: Si es True, no distingue entre mayúsculas y minúsculas
        
    Returns:
        bool: True si el texto es un palíndromo, False en caso contrario
    """
    if not texto:
        return False
    
    # Aplicar transformaciones según los parámetros
    procesado = texto
    
    if ignorar_mayusculas:
        procesado = procesado.lower()
    
    if ignorar_acentos:
        # Reemplazar caracteres acentuados
        reemplazos = {
            'á': 'a', 'é': 'e', 'í': 'i', 'ó': 'o', 'ú': 'u',
            'Á': 'a', 'É': 'e', 'Í': 'i', 'Ó': 'o', 'Ú': 'u',
            'ü': 'u', 'Ü': 'u', 'ñ': 'n', 'Ñ': 'n'
        }
        for original, reemplazo in reemplazos.items():
            procesado = procesado.replace(original, reemplazo)
    
    # Eliminar caracteres que no sean letras ni números
    procesado = re.sub(r'[^a-z0-9]', '', procesado)
    
    if not ignorar_espacios:
        # Si no se ignoran los espacios, verificar que el texto original no tenga espacios
        # ya que ya los eliminamos en el paso anterior
        if ' ' in texto:
            return False
    
    # Comparar con su versión invertida
    return procesado == procesado[::-1]

def contar_palabras(texto: str) -> int:
    """
    Cuenta el número de palabras en un texto.
    
    Args:
        texto: Texto a analizar
        
    Returns:
        int: Número de palabras en el texto
    """
    if not texto.strip():
        return 0
    
    # Dividir por espacios en blanco y filtrar cadenas vacías
    palabras = [palabra for palabra in texto.split() if palabra]
    return len(palabras)

__all__ = ['invertir_texto', 'contar_vocales', 'es_palindromo', 'contar_palabras']

"""
Funciones para manejo de archivos.
"""
import os
import json
import csv
from typing import Any, Dict, List, Union, Optional
from pathlib import Path

def leer_archivo(ruta: Union[str, Path], modo: str = 'r', 
                codificacion: str = 'utf-8') -> str:
    """
    Lee el contenido de un archivo de texto.
    
    Args:
        ruta: Ruta al archivo
        modo: Modo de apertura ('r' para texto, 'rb' para binario)
        codificacion: Codificación del archivo (solo para modo texto)
        
    Returns:
        str: Contenido del archivo
        
    Raises:
        FileNotFoundError: Si el archivo no existe
        IOError: Si hay un error al leer el archivo
    """
    ruta_obj = Path(ruta) if isinstance(ruta, str) else ruta
    
    if 'b' in modo:
        with open(ruta_obj, modo) as archivo:
            return archivo.read()
    else:
        with open(ruta_obj, modo, encoding=codificacion) as archivo:
            return archivo.read()

escribir_archivo(ruta: Union[str, Path], contenido: Any, 
                modo: str = 'w', 
                codificacion: str = 'utf-8',
                formato: Optional[str] = None) -> None:
    """
    Escribe contenido en un archivo.
    
    Args:
        ruta: Ruta al archivo
        contenido: Contenido a escribir (str, bytes o objeto serializable)
        modo: Modo de apertura ('w' para sobrescribir, 'a' para anexar)
        codificacion: Codificación del archivo (solo para modo texto)
        formato: Formato de salida ('json', 'csv', None para texto plano)
        
    Raises:
        IOError: Si hay un error al escribir el archivo
        ValueError: Si el formato no es soportado
    """
    ruta_obj = Path(ruta) if isinstance(ruta, str) else ruta
    
    # Crear directorio si no existe
    ruta_obj.parent.mkdir(parents=True, exist_ok=True)
    
    if formato == 'json':
        with open(ruta_obj, modo, encoding=codificacion) as archivo:
            json.dump(contenido, archivo, indent=2, ensure_ascii=False)
    elif formato == 'csv':
        with open(ruta_obj, modo, newline='', encoding=codificacion) as archivo:
            if isinstance(contenido, list) and len(contenido) > 0:
                writer = csv.DictWriter(archivo, fieldnames=contenido[0].keys())
                writer.writeheader()
                writer.writerows(contenido)
            else:
                raise ValueError("El contenido debe ser una lista de diccionarios para formato CSV")
    else:
        # Modo texto plano
        if not isinstance(contenido, (str, bytes, bytearray)):
            contenido = str(contenido)
            
        if 'b' in modo and isinstance(contenido, str):
            contenido = contenido.encode(codificacion)
        
        with open(ruta_obj, 'b' + modo if 'b' not in modo else modo) as archivo:
            archivo.write(contenido)

def listar_directorio(ruta: Union[str, Path] = '.', 
                     mostrar_ocultos: bool = False) -> List[Dict[str, Any]]:
    """
    Lista el contenido de un directorio.
    
    Args:
        ruta: Ruta al directorio
        mostrar_ocultos: Si es True, incluye archivos y carpetas ocultas
        
    Returns:
        List[Dict[str, Any]]: Lista de diccionarios con información de cada elemento
    """
    ruta_obj = Path(ruta) if isinstance(ruta, str) else ruta
    
    if not ruta_obj.exists():
        raise FileNotFoundError(f"El directorio no existe: {ruta_obj}")
    
    if not ruta_obj.is_dir():
        raise NotADirectoryError(f"La ruta no es un directorio: {ruta_obj}")
    
    elementos = []
    
    for item in ruta_obj.iterdir():
        # Saltar archivos ocultos si no se solicitan
        if not mostrar_ocultos and item.name.startswith('.'):
            continue
            
        try:
            stat = item.stat()
            elementos.append({
                'nombre': item.name,
                'ruta': str(item.absolute()),
                'es_archivo': item.is_file(),
                'es_directorio': item.is_dir(),
                'tamano_bytes': stat.st_size if item.is_file() else 0,
                'fecha_modificacion': stat.st_mtime,
                'permisos': oct(stat.st_mode)[-3:]
            })
        except (OSError, PermissionError):
            # Ignorar archivos a los que no se puede acceder
            continue
    
    return elementos

__all__ = ['leer_archivo', 'escribir_archivo', 'listar_directorio']

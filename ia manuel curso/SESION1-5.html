Contenido Teórico

1. Introducción a las Funciones

Una función es un bloque de código organizado y reutilizable que realiza una acción específica. Las funciones proporcionan una mejor modularidad para tu aplicación y un alto grado de reutilización de código.

1.1 Definición de funciones

En Python, una función se define utilizando la palabra clave def:

def saludar():
    """Esta función imprime un saludo"""
    print("¡Hola, bienvenido al curso de Python!")

# Llamada a la función
saludar()

1.2 Parámetros y argumentos

Las funciones pueden recibir parámetros:

def saludar_personalizado(nombre):
    """Saluda a una persona por su nombre"""
    print(f"¡Hola {nombre}, bienvenido al curso de Python!")

saludar_personalizado("Juan")

1.3 Retorno de valores

Las funciones pueden devolver valores usando la palabra clave return:

def sumar(a, b):
    """Devuelve la suma de dos números"""
    return a + b

resultado = sumar(5, 3)
print(f"La suma es: {resultado}")

2. Módulos en Python

Un módulo es un archivo que contiene definiciones y declaraciones de Python. El nombre del módulo es el nombre del archivo con el sufijo .py.

2.1 Crear un módulo

Puedes crear tu propio módulo guardando tu código en un archivo con extensión .py.

2.2 Importar un módulo

# Importar todo el módulo
import math

# Usar funciones del módulo
raiz = math.sqrt(16)  # 4.0

2.3 Importar funciones específicas

# Importar solo lo necesario
from math import sqrt, pi

# Usar directamente las funciones importadas
area = pi * (5 ** 2)  # 78.53981633974483

Ejercicios Prácticos

Ejercicio 1: Calculadora de IMC

Crea una función llamada calcular_imc que reciba el peso (en kg) y la altura (en metros) de una persona y devuelva su Índice de Masa Corporal (IMC).

def calcular_imc(peso, altura):
    """
    Calcula el Índice de Masa Corporal (IMC)
    
    Args:
        peso (float): Peso en kilogramos
        altura (float): Altura en metros
        
    Returns:
        float: Valor del IMC redondeado a 2 decimales
    """
    imc = peso / (altura ** 2)
    return round(imc, 2)

# Ejemplo de uso
imc = calcular_imc(70, 1.75)
print(f"Tu IMC es: {imc}")

# Interpretación del IMC
if imc < 18.5:
    print("Bajo peso")
elif 18.5 <= imc < 25:
    print("Peso normal")
elif 25 <= imc < 30:
    print("Sobrepeso")
else:
    print("Obesidad")

Ejercicio 2: Módulo de utilidades matemáticas

Crea un módulo llamado matematicas.py con funciones para calcular el área de diferentes figuras geométricas. Luego impórtalo y úsalo.

1. Primero, crea un archivo llamado matematicas.py con el siguiente contenido:

"""Módulo con funciones matemáticas útiles"""
import math

def area_circulo(radio):
    """Calcula el área de un círculo"""
    return math.pi * (radio ** 2)

def area_rectangulo(base, altura):
    """Calcula el área de un rectángulo"""
    return base * altura

def area_triangulo(base, altura):
    """Calcula el área de un triángulo"""
    return (base * altura) / 2

def es_primo(numero):
    """Determina si un número es primo"""
    if numero < 2:
        return False
    for i in range(2, int(math.sqrt(numero)) + 1):
        if numero % i == 0:
            return False
    return True

2. Luego, en otro archivo o en el intérprete interactivo:

# Importar el módulo
import matematicas

# Usar las funciones del módulo
print(f"Área del círculo: {matematicas.area_circulo(5):.2f}")
print(f"Área del rectángulo: {matematicas.area_rectangulo(4, 6)}")
print(f"¿El número 17 es primo? {matematicas.es_primo(17)}")

# O importar solo lo necesario
from matematicas import area_triangulo
print(f"Área del triángulo: {area_triangulo(3, 4)}")

Ejercicio 3: Generador de contraseñas

Crea una función que genere contraseñas seguras con letras mayúsculas, minúsculas, números y símbolos.

import random
import string

def generar_contraseña(longitud=12):
    """
    Genera una contraseña segura
    
    Args:
        longitud (int): Longitud de la contraseña (mínimo 8 caracteres)
        
    Returns:
        str: Contraseña generada
    """
    if longitud < 8:
        longitud = 8  # Longitud mínima recomendada
    
    # Definir los conjuntos de caracteres
    minusculas = string.ascii_lowercase
    mayusculas = string.ascii_uppercase
    digitos = string.digits
    simbolos = '!@#$%^&*()_+-=[]{}|;:,.<>?'
    
    # Asegurar al menos un carácter de cada tipo
    contraseña = [
        random.choice(minusculas),
        random.choice(mayusculas),
        random.choice(digitos),
        random.choice(simbolos)
    ]
    
    # Completar el resto de la contraseña
    todos_caracteres = minusculas + mayusculas + digitos + simbolos
    contraseña.extend(random.choice(todos_caracteres) for _ in range(longitud - 4))
    
    # Mezclar los caracteres
    random.shuffle(contraseña)
    
    return ''.join(contraseña)

# Ejemplo de uso
print("Contraseña generada:", generar_contraseña(16))

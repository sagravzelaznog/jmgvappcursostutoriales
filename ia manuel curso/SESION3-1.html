Contenido Teórico

1. ¿Qué es Machine Learning?

El Machine Learning (Aprendizaje Automático) es una rama de la inteligencia artificial que se enfoca en desarrollar sistemas que pueden aprender de los datos y mejorar su rendimiento en tareas específicas sin ser programados explícitamente.

Según Arthur Samuel (1959), el aprendizaje automático es el "campo de estudio que le da a las computadoras la capacidad de aprender sin estar explícitamente programadas".

El aprendizaje automático se diferencia de la programación tradicional en que, en lugar de escribir reglas explícitas para resolver un problema, el algoritmo aprende patrones a partir de los datos de ejemplo.

2. Tipos de Aprendizaje Automático

Aprendizaje Supervisado

El algoritmo aprende a partir de datos etiquetados, donde cada ejemplo de entrenamiento incluye tanto las características de entrada como la salida deseada.

Ejemplos:

  • Clasificación de correos electrónicos (spam/no spam)
  • Reconocimiento de dígitos escritos a mano
  • Predicción de precios de viviendas

Aprendizaje No Supervisado

El algoritmo trabaja con datos sin etiquetar y debe encontrar patrones o estructuras ocultas en los datos.

Ejemplos:

  • Segmentación de clientes
  • Detección de anomalías
  • Reducción de dimensionalidad

Aprendizaje por Refuerzo

Un agente aprende a realizar tareas mediante la interacción con un entorno, recibiendo recompensas o castigos por sus acciones.

Ejemplos:

  • Juegos (AlphaGo, Dota 2 bots)
  • Robótica
  • Sistemas de recomendación

3. Flujo de Trabajo en Machine Learning

Proceso típico en un proyecto de ML:

  1. Recopilación de datos: Obtener datos relevantes para el problema.
  2. Preprocesamiento: Limpiar, normalizar y transformar los datos.
  3. Análisis exploratorio: Entender los datos mediante visualizaciones y estadísticas.
  4. Selección de características: Identificar las variables más relevantes.
  5. Selección del modelo: Elegir el algoritmo de ML apropiado.
  6. Entrenamiento: Ajustar el modelo a los datos de entrenamiento.
  7. Evaluación: Medir el rendimiento del modelo con datos no vistos.
  8. Optimización: Ajustar hiperparámetros para mejorar el rendimiento.
  9. Despliegue: Implementar el modelo en producción.
  10. Monitoreo y mantenimiento: Asegurar que el modelo siga funcionando correctamente.

4. Aplicaciones del Machine Learning

Diagnóstico Médico

Ayuda a los médicos a diagnosticar enfermedades mediante el análisis de imágenes médicas y registros de pacientes.

Vehículos Autónomos

Permite a los coches percibir su entorno y tomar decisiones de conducción de forma autónoma.

Recomendaciones

Sistemas de recomendación en plataformas como Netflix, Amazon y Spotify.

Procesamiento de Lenguaje Natural

Asistentes virtuales, traducción automática y análisis de sentimientos.

Análisis Financiero

Detección de fraudes, evaluación de riesgos y trading algorítmico.

Visión por Computadora

Reconocimiento facial, detección de objetos y clasificación de imágenes.

2. Primer Ejemplo: Clasificación con el Conjunto de Datos Iris

Vamos a implementar nuestro primer modelo de Machine Learning utilizando el conjunto de datos Iris, que es un conjunto clásico en la comunidad de ML.

# Importar bibliotecas necesarias
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from sklearn.neighbors import KNeighborsClassifier
from sklearn.metrics import classification_report, confusion_matrix, accuracy_score

# Cargar el conjunto de datos Iris
iris = load_iris()
X = iris.data  # Características
y = iris.target  # Etiquetas
feature_names = iris.feature_names
target_names = iris.target_names

# Crear un DataFrame para visualización
df = pd.DataFrame(X, columns=feature_names)
df['target'] = y
df['species'] = df['target'].map({i: name for i, name in enumerate(target_names)})

# Visualizar los primeros registros
print("\nPrimeras filas del conjunto de datos:")
print(df.head())

# Estadísticas descriptivas
print("\nEstadísticas descriptivas:")
print(df.describe())

# Visualización de los datos
plt.figure(figsize=(12, 6))

# Gráfico de dispersión de las características
plt.subplot(1, 2, 1)
sns.scatterplot(x='sepal length (cm)', y='sepal width (cm)', hue='species', data=df, palette='viridis')
plt.title('Longitud vs Ancho del Sépalo')

plt.subplot(1, 2, 2)
sns.scatterplot(x='petal length (cm)', y='petal width (cm)', hue='species', data=df, palette='viridis')
plt.title('Longitud vs Ancho del Pétalo')

plt.tight_layout()
plt.show()

# Dividir los datos en conjuntos de entrenamiento y prueba
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42, stratify=y)

# Estandarizar las características
scaler = StandardScaler()
X_train_scaled = scaler.fit_transform(X_train)
X_test_scaled = scaler.transform(X_test)

# Crear y entrenar el modelo K-Vecinos más Cercanos
knn = KNeighborsClassifier(n_neighbors=3)
knn.fit(X_train_scaled, y_train)

# Realizar predicciones
y_pred = knn.predict(X_test_scaled)

# Evaluar el modelo
accuracy = accuracy_score(y_test, y_pred)
print(f"\nPrecisión del modelo: {accuracy:.2f}")

# Mostrar el informe de clasificación
print("\nInforme de Clasificación:")
print(classification_report(y_test, y_pred, target_names=target_names))

# Matriz de confusión
plt.figure(figsize=(8, 6))
cm = confusion_matrix(y_test, y_pred)
sns.heatmap(cm, annot=True, fmt='d', cmap='Blues', 
            xticklabels=target_names, 
            yticklabels=target_names)
plt.title('Matriz de Confusión')
plt.xlabel('Predicción')
plt.ylabel('Verdadero')
plt.show()

  3. ¿Qué patrones puedes observar en la matriz de confusión?

Análisis de la Solución:

Ejercicio 1: Exploración de Hiperparámetros

Modifica el código anterior para probar diferentes valores de k (1, 3, 5, 7, 9, 11) en el clasificador KNN. Crea una gráfica que muestre la precisión del modelo en el conjunto de entrenamiento y prueba para cada valor de k. ¿Qué observas?

import numpy as np
import matplotlib.pyplot as plt
from sklearn.neighbors import KNeighborsClassifier
from sklearn.metrics import accuracy_score

# Probar diferentes valores de k
k_values = list(range(1, 21, 2))
train_scores = []
test_scores = []

for k in k_values:
    # Crear y entrenar el modelo
    knn = KNeighborsClassifier(n_neighbors=k)
    knn.fit(X_train_scaled, y_train)
    
    # Calcular precisión en entrenamiento y prueba
    train_pred = knn.predict(X_train_scaled)
    test_pred = knn.predict(X_test_scaled)
    
    train_scores.append(accuracy_score(y_train, train_pred))
    test_scores.append(accuracy_score(y_test, test_pred))

# Graficar los resultados
plt.figure(figsize=(10, 6))
plt.plot(k_values, train_scores, 'o-', label='Precisión en Entrenamiento')
plt.plot(k_values, test_scores, 'o-', label='Precisión en Prueba')
plt.title('Precisión vs. Valor de k en KNN')
plt.xlabel('Valor de k')
plt.ylabel('Precisión')
plt.xticks(k_values)
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

# Encontrar el mejor valor de k
best_k = k_values[np.argmax(test_scores)]
print(f"\nMejor valor de k: {best_k} con precisión de prueba: {max(test_scores):.4f}")

Ejercicio 2: Comparación de Algoritmos

Compara el rendimiento de al menos tres algoritmos de clasificación diferentes (por ejemplo, KNN, Árbol de Decisión y Regresión Logística) en el conjunto de datos Iris. Crea una tabla que muestre la precisión, precisión, recuperación y puntuación F1 para cada modelo.

from sklearn.tree import DecisionTreeClassifier
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import precision_recall_fscore_support

# Inicializar los modelos
models = {
    'K-Vecinos más Cercanos': KNeighborsClassifier(n_neighbors=3),
    'Árbol de Decisión': DecisionTreeClassifier(random_state=42),
    'Regresión Logística': LogisticRegression(max_iter=1000, random_state=42)
}

# Entrenar y evaluar cada modelo
results = []

for name, model in models.items():
    # Entrenar el modelo
    model.fit(X_train_scaled, y_train)
    
    # Realizar predicciones
    y_pred = model.predict(X_test_scaled)
    
    # Calcular métricas
    accuracy = accuracy_score(y_test, y_pred)
    precision, recall, f1, _ = precision_recall_fscore_support(y_test, y_pred, average='weighted')
    
    results.append({
        'Modelo': name,
        'Precisión': accuracy,
        'Precisión (weighted)': precision,
        'Recuperación (weighted)': recall,
        'F1 (weighted)': f1
    })

# Mostrar resultados en una tabla
df_results = pd.DataFrame(results)
print("\nComparación de Modelos:")
print(df_results.to_string(index=False))

# Visualizar comparación
plt.figure(figsize=(12, 6))
metrics = ['Precisión', 'Precisión (weighted)', 'Recuperación (weighted)', 'F1 (weighted)']
x = np.arange(len(metrics))
width = 0.2

for i, (model_name, _) in enumerate(models.items()):
    values = [results[i][m] for m in metrics]
    plt.bar(x + i*width, values, width, label=model_name)

plt.xlabel('Métricas')
plt.ylabel('Puntuación')
plt.title('Comparación de Modelos de Clasificación')
plt.xticks(x + width, metrics, rotation=45, ha='right')
plt.legend()
plt.tight_layout()
plt.show()

Conceptos Clave:

Aprendizaje Supervisado

Modelos que aprenden a partir de ejemplos etiquetados para hacer predicciones sobre datos no vistos.

K-Vecinos más Cercanos (KNN)

Algoritmo de clasificación que asigna a cada punto la clase más común entre sus k vecinos más cercanos.

Conjunto de Entrenamiento/Prueba

División de los datos para entrenar el modelo y evaluar su rendimiento en datos no vistos.

Estandarización

Proceso de escalar las características para que tengan media 0 y desviación estándar 1.

Matriz de Confusión

Tabla que muestra el rendimiento de un modelo de clasificación comparando las predicciones con los valores reales.

Métricas de Evaluación

Métricas como precisión, recuperación y F1 que nos permiten medir el rendimiento de un modelo.

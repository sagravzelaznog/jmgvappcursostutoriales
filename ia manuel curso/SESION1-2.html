Contenido Teórico

¿Qué son las variables?

En programación, una variable es un espacio en la memoria de la computadora donde podemos almacenar datos. Piensa en ellas como cajas etiquetadas donde guardamos información para usarla más tarde.

Curiosidad: En Python, las variables no necesitan una declaración de tipo explícita. Python infiere el tipo de dato automáticamente.

Reglas para nombrar variables:

  • Pueden contener letras, números y guiones bajos (_)
  • No pueden comenzar con un número
  • Son sensibles a mayúsculas y minúsculas (edad ≠ Edad ≠ EDAD)
  • No pueden ser palabras reservadas de Python (como if, for, while, etc.)

Tipos de Datos Básicos

1. Números

Enteros (int)

Números sin parte decimal. Ejemplos: 5, -3, 0, 42

edad = 25
puntuacion = -10
anio = 2023
Punto flotante (float)

Números con parte decimal. Ejemplos: 3.14, -0.001, 2.0

pi = 3.1416
precio = 19.99
temperatura = -5.5

2. Texto (str)

Cadenas de caracteres entre comillas simples o dobles.

nombre = "Juan"
mensaje = 'Hola, ¿cómo estás?'
direccion = "Calle Falsa 123"

3. Booleanos (bool)

Representan valores de verdad: True o False (Verdadero o Falso).

es_mayor = True
tiene_descuento = False
es_estudiante = True

4. Colecciones (las veremos en detalle más adelante)

  • Listas (list): Colección ordenada y mutable [1, 2, 3]
  • Tuplas (tuple): Colección ordenada e inmutable (1, 2, 3)
  • Diccionarios (dict): Pares clave-valor {"nombre": "Ana", "edad": 25}
  • Conjuntos (set): Colección no ordenada de elementos únicos {1, 2, 3}

Operaciones con Variables

Operaciones numéricas

# Suma
resultado = 5 + 3  # 8

# Resta
diferencia = 10 - 4  # 6

# Multiplicación
producto = 6 * 7  # 42

# División
cociente = 15 / 3  # 5.0 (siempre devuelve float)

# División entera
cociente_entero = 15 // 4  # 3

# Módulo (resto de la división)
resto = 15 % 4  # 3

# Potencia
potencia = 2 ** 3  # 8

Operaciones con cadenas

# Concatenación
saludo = "Hola, " + "mundo!"  # "Hola, mundo!"

# Repetición
risas = "ja" * 3  # "jajaja"

# Longitud de una cadena
longitud = len("Python")  # 6

Conversión entre tipos

# De cadena a entero
numero = int("42")  # 42

# De entero a cadena
texto = str(42)  # "42"

# De entero a flotante
decimal = float(5)  # 5.0

# De flotante a entero (trunca los decimales)
entero = int(3.99)  # 3 (no redondea, solo quita la parte decimal)

Actividad Práctica

1. Ejercicio: Calculadora Básica

Crea un programa que pida al usuario dos números y muestre el resultado de las operaciones básicas.

Pista
Usa input() para obtener los números (recuerda convertirlos a int() o float()) y luego muestra los resultados con print().

2. Ejercicio: Información Personal

Crea un programa que pida al usuario su nombre, edad y ciudad, y muestre un mensaje personalizado.

Pista
Usa input() para cada dato y luego concaténalos en un mensaje con + o usando f-strings (Python 3.6+).

3. Ejercicio: Conversor de Unidades

Crea un programa que convierta kilómetros a millas (1 km = 0.621371 millas).

Ejercicios Adicionales

Ejercicio 1: Calculadora de IMC

Escribe un programa que calcule el Índice de Masa Corporal (IMC) usando la fórmula: IMC = peso / (altura^2).

Donde el peso está en kg y la altura en metros.

Ejercicio 2: Calculadora de Propinas

Crea un programa que calcule el monto total a pagar en un restaurante, incluyendo una propina del 15% sobre el total de la cuenta.

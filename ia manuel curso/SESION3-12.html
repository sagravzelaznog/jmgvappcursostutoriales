Introducción a la Evaluación de Modelos

La evaluación de modelos es fundamental para entender el rendimiento y la capacidad de generalización de nuestros modelos de aprendizaje automático.

¿Por qué evaluar modelos?

  • Medir el rendimiento predictivo
  • Comparar diferentes modelos
  • Detectar sobreajuste (overfitting)
  • Tomar decisiones informadas

Métricas de Evaluación

Clasificación

  • Exactitud (Accuracy)
  • Precisión y Recall
  • F1-Score
  • Matriz de confusión
  • Curva ROC y AUC

Regresión

  • Error Cuadrático Medio (MSE)
  • Error Absoluto Medio (MAE)
  • R² (R cuadrado)
  • Error Porcentual Absoluto Medio (MAPE)

Matriz de Confusión

Predicción Positiva Predicción Negativa
Real Positivo Verdaderos Positivos (VP) Falsos Negativos (FN)
Real Negativo Falsos Positivos (FP) Verdaderos Negativos (VN)

Técnicas de Validación

Train-Test Split

División simple de los datos en conjuntos de entrenamiento y prueba.

from sklearn.model_selection import train_test_split

X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=42
)
n

Validación Cruzada

K-Fold Cross-Validation para una evaluación más robusta.

from sklearn.model_selection import cross_val_score

scores = cross_val_score(
    model, X, y, cv=5, scoring='accuracy'
)
print(f"Precisión media: {scores.mean():.2f} ± {scores.std():.2f}")

Ejercicios Prácticos

Ejercicio 1: Evaluación de Clasificador

Utiliza el conjunto de datos de cáncer de mama de scikit-learn:

  1. Entrena un clasificador (ej. Random Forest)
  2. Calcula las métricas de evaluación
  3. Dibuja la matriz de confusión
  4. Grafica la curva ROC

Puedes usar load_breast_cancer() de sklearn.datasets para cargar los datos.

  2. Árbol de Decisión

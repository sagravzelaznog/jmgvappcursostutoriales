Contenido Teórico

¿Qué es un sistema de ecuaciones lineales?

Un sistema de ecuaciones lineales es un conjunto de ecuaciones lineales que involucran el mismo conjunto de variables. Por ejemplo:

\[ \begin{cases} 2x + 3y = 5 \\ 4x - y = 6 \end{cases} \]

Donde x e y son las variables que queremos encontrar.

Dato clave: Los sistemas de ecuaciones lineales son fundamentales en IA, especialmente en algoritmos como regresión lineal, redes neuronales y optimización.

Representación matricial

Todo sistema de ecuaciones lineales puede representarse en forma matricial como AX = B, donde:

  • A es la matriz de coeficientes
  • X es el vector de incógnitas
  • B es el vector de términos independientes

Por ejemplo, el sistema anterior se puede escribir como:

\[ \begin{bmatrix} 2 & 3 \\ 4 & -1 \end{bmatrix} \begin{bmatrix} x \\ y \end{bmatrix} = \begin{bmatrix} 5 \\ 6 \end{bmatrix} \]

Métodos de solución

1. Método de sustitución

Consiste en despejar una variable en una ecuación y sustituirla en las demás.

2. Método de eliminación de Gauss

Transforma la matriz aumentada a su forma escalonada mediante operaciones elementales de fila.

3. Regla de Cramer

Utiliza determinantes para encontrar la solución del sistema, aplicable solo cuando el determinante de la matriz de coeficientes es distinto de cero.

4. Método de la matriz inversa

Si la matriz A es invertible, la solución es X = A-1B.

Importante: No todos los sistemas tienen solución. Un sistema puede ser:

  • Compatible determinado: Tiene una única solución
  • Compatible indeterminado: Tiene infinitas soluciones
  • Incompatible: No tiene solución

Actividad Práctica

1. Resolución de sistemas con NumPy

Vamos a resolver el siguiente sistema de ecuaciones usando Python y NumPy:

\[ \begin{cases} 2x + 3y = 5 \\ 4x - y = 6 \end{cases} \]

Código en Python:

import numpy as np

# Definir la matriz de coeficientes
A = np.array([[2, 3], [4, -1]])

# Definir el vector de términos independientes
B = np.array([5, 6])

# Resolver el sistema
solucion = np.linalg.solve(A, B)

print(f"La solución es: x = {solucion[0]:.2f}, y = {solucion[1]:.2f}")
Pista: ¿Qué hace np.linalg.solve?

La función np.linalg.solve resuelve un sistema de ecuaciones lineales de la forma AX = B, donde A es la matriz de coeficientes y B es el vector de términos independientes. Retorna el vector solución X.

2. Ejercicio práctico

Resuelve el siguiente sistema de ecuaciones usando el código anterior:

\[ \begin{cases} 3x + 2y - z = 1 \\ 2x - 2y + 4z = -2 \\ -x + 0.5y - z = 0 \end{cases} \] 
import numpy as np

A = np.array([
    [3, 2, -1],
    [2, -2, 4],
    [-1, 0.5, -1]
])

B = np.array([1, -2, 0])

solucion = np.linalg.solve(A, B)
print(f"Solución: x = {solucion[0]:.2f}, y = {solucion[1]:.2f}, z = {solucion[2]:.2f}")

La solución debería ser aproximadamente: x = 1.00, y = -2.00, z = -2.00

Resumen

  • Los sistemas de ecuaciones lineales son fundamentales en IA y Machine Learning.
  • Pueden representarse en forma matricial como AX = B.
  • Existen varios métodos para resolverlos, incluyendo sustitución, eliminación de Gauss, regla de Cramer y matriz inversa.
  • En Python, podemos usar NumPy para resolver sistemas de ecuaciones de manera eficiente.
  • Es importante verificar si un sistema tiene solución única, infinitas soluciones o ninguna solución.

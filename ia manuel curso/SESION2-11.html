Contenido Teórico

1. Repaso de Probabilidad Condicional

La probabilidad condicional es la probabilidad de que ocurra un evento A dado que ya ocurrió un evento B.

Fórmula de Probabilidad Condicional

\[P(A|B) = \frac{P(A \cap B)}{P(B)}\]

Donde:

  • \(P(A|B)\): Probabilidad de A dado B
  • \(P(A \cap B)\): Probabilidad de A y B ocurriendo juntos
  • \(P(B)\): Probabilidad de B

2. Teorema de Bayes

Fórmula del Teorema de Bayes

\[P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}\]

Donde:

  • \(P(A|B)\): Probabilidad a posteriori (nuestra hipótesis después de ver la evidencia)
  • \(P(B|A)\): Verosimilitud (probabilidad de la evidencia dada la hipótesis)
  • \(P(A)\): Probabilidad a priori (nuestra creencia inicial)
  • \(P(B)\): Evidencia (probabilidad total de la evidencia)

4. Naive Bayes

El clasificador Naive Bayes es un algoritmo de aprendizaje automático basado en el Teorema de Bayes con el supuesto de independencia entre predictores.

Clasificador Naive Bayes

\[P(y|X) = \frac{P(X|y) \cdot P(y)}{P(X)}\]

Donde:

  • \(y\): Clase de la variable objetivo
  • \(X\): Conjunto de características
  • Bajo el supuesto de independencia: \(P(X|y) = P(x_1|y) \cdot P(x_2|y) \cdot ... \cdot P(x_n|y)\)

Vamos a implementar el Teorema de Bayes para resolver problemas de probabilidad condicional.

def teorema_bayes(p_a, p_b_dado_a, p_b_dado_no_a=None):
    """
    Calcula P(A|B) usando el Teorema de Bayes
    
    Parámetros:
    p_a: Probabilidad a priori P(A)
    p_b_dado_a: Probabilidad de B dado A, P(B|A)
    p_b_dado_no_a: Probabilidad de B dado no A, P(B|¬A)
    
    Retorna:
    Probabilidad a posteriori P(A|B)
    """
    if p_b_dado_no_a is None:
        # Si no se proporciona P(B|¬A), asumir que P(B) = P(B|A)P(A) + P(B|¬A)P(¬A) = 1
        p_b = p_b_dado_a * p_a + (1 - p_a) * 0.5  # Asumir 0.5 como valor por defecto
    else:
        p_b = p_b_dado_a * p_a + p_b_dado_no_a * (1 - p_a)
    
    p_a_dado_b = (p_b_dado_a * p_a) / p_b
    return p_a_dado_b

2. Implementación del Clasificador Naive Bayes

Vamos a implementar un clasificador Naive Bayes desde cero y compararlo con la implementación de scikit-learn.

import numpy as np
from collections import defaultdict

class NaiveBayes:
    def __init__(self, alpha=1.0):
        self.alpha = alpha  # Parámetro de suavizado de Laplace
        self.classes = None
        self.class_priors = {}
        self.feature_probs = {}
    
    def fit(self, X, y):
        """
        Entrena el clasificador Naive Bayes
        
        Parámetros:
        X: array-like de forma (n_muestras, n_caracteristicas)
        y: array-like de forma (n_muestras,)
        """
        self.classes = np.unique(y)
        n_samples, n_features = X.shape
        
        # Calcular las probabilidades a priori de cada clase
        for c in self.classes:
            self.class_priors[c] = np.sum(y == c) / n_samples
        
        # Calcular las probabilidades condicionales P(x_i|y)
        for c in self.classes:
            # Filtrar muestras de la clase actual
            X_c = X[y == c]
            self.feature_probs[c] = {}
            
            # Para cada característica
            for i in range(n_features):
                # Contar ocurrencias de cada valor
                values, counts = np.unique(X_c[:, i], return_counts=True)
                total = np.sum(counts)
                
                # Calcular probabilidades con suavizado de Laplace
                self.feature_probs[c][i] = {}
                for v, cnt in zip(values, counts):
                    self.feature_probs[c][i][v] = (cnt + self.alpha) / (total + self.alpha * len(np.unique(X[:, i])))
    
    def predict_proba(self, X):
        """
        Predice las probabilidades para cada clase
        
        Retorna:
        array de forma (n_muestras, n_clases) con las probabilidades
        """
        n_samples = X.shape[0]
        n_classes = len(self.classes)
        probs = np.zeros((n_samples, n_classes))
        
        for i in range(n_samples):
            for j, c in enumerate(self.classes):
                # Inicializar con el logaritmo de la probabilidad a priori
                log_prob = np.log(self.class_priors[c])
                
                # Multiplicar por las probabilidades condicionales (sumar en escala logarítmica)
                for k, x in enumerate(X[i]):
                    # Si el valor se ha visto en el entrenamiento, usar su probabilidad
                    if x in self.feature_probs[c][k]:
                        log_prob += np.log(self.feature_probs[c][k][x])
                    else:
                        # Manejar valores no vistos (usar una probabilidad pequeña)
                        log_prob += np.log(self.alpha / (sum([len(self.feature_probs[cls][k]) for cls in self.classes]) * self.alpha))
                
                # Convertir de vuelta a probabilidad (exponenciar)
                probs[i, j] = np.exp(log_prob)
            
            # Normalizar las probabilidades
            probs[i] = probs[i] / np.sum(probs[i])
        
        return probs
    
    def predict(self, X):
        """
        Predice la clase para cada muestra
        
        Retorna:
        array de forma (n_muestras,) con las clases predichas
        """
        probs = self.predict_proba(X)
        return self.classes[np.argmax(probs, axis=1)]

# Ejemplo de uso con un conjunto de datos de juguete
def ejemplo_clasificacion():
    from sklearn.datasets import load_iris
    from sklearn.model_selection import train_test_split
    from sklearn.metrics import accuracy_score, classification_report
    from sklearn.naive_bayes import GaussianNB
    
    # Cargar el conjunto de datos Iris
    data = load_iris()
    X, y = data.data, data.target
    
    # Discretizar características (para nuestro Naive Bayes simple)
    X_discreto = np.digitize(X, bins=np.linspace(X.min(), X.max(), 5))
    
    # Dividir en conjuntos de entrenamiento y prueba
    X_train, X_test, y_train, y_test = train_test_split(
        X_discreto, y, test_size=0.3, random_state=42, stratify=y
    )
    
    # Entrenar nuestro clasificador
    nb_custom = NaiveBayes(alpha=1.0)
    nb_custom.fit(X_train, y_train)
    
    # Predecir en el conjunto de prueba
    y_pred_custom = nb_custom.predict(X_test)
    
    # Entrenar el clasificador de scikit-learn para comparar
    # Nota: Usamos GaussianNB ya que nuestro conjunto de datos tiene características continuas
    nb_sklearn = GaussianNB()
    nb_sklearn.fit(X, y)  # Usamos los datos originales (sin discretizar)
    y_pred_sklearn = nb_sklearn.predict(X_test)
    
    # Calcular y mostrar métricas
    print("=== Nuestro Clasificador Naive Bayes ===")
    print(f"Precisión: {accuracy_score(y_test, y_pred_custom):.4f}")
    print("\nReporte de clasificación:")
    print(classification_report(y_test, y_pred_custom, target_names=data.target_names))
    
    print("\n=== Comparación con scikit-learn GaussianNB ===")
    print(f"Precisión: {accuracy_score(y_test, y_pred_sklearn):.4f}")
    print("\nReporte de clasificación:")
    print(classification_report(y_test, y_pred_sklearn, target_names=data.target_names))

def preprocesar_texto(correos):
    """
    Preprocesa el texto para el análisis
    """
    # En un caso real, aquí se haría limpieza más avanzada
    # (eliminación de stopwords, stemming, lematización, etc.)
    return [texto.lower() for texto in correos]

def entrenar_filtro_spam():
    """
    Entrena y evalúa un filtro de spam usando Naive Bayes
    """
    # Cargar datos de ejemplo
    correos, y = cargar_datos_ejemplo()
    
    # Preprocesar texto
    correos_limpios = preprocesar_texto(correos)
    
    # Convertir texto a vectores de conteo de palabras
    vectorizador = CountVectorizer(binary=True)
    X = vectorizador.fit_transform(correos_limpios)
    
    # Dividir en conjuntos de entrenamiento y prueba (usando validación simple)
    # En un caso real, se usaría train_test_split de scikit-learn
    X_train, X_test = X[:15], X[15:]
    y_train, y_test = y[:15], y[15:]
    
    # Entrenar el clasificador Naive Bayes
    clasificador = MultinomialNB()
    clasificador.fit(X_train, y_train)
    
    # Hacer predicciones
    y_pred = clasificador.predict(X_test)
    
    # Evaluar el rendimiento
    print("=== Evaluación del Filtro de Spam ===")
    print(f"Precisión: {accuracy_score(y_test, y_pred):.2f}")
    print("\nReporte de clasificación:")
    print(classification_report(y_test, y_pred, target_names=['Ham', 'Spam']))
    
    # Mostrar matriz de confusión
    cm = confusion_matrix(y_test, y_pred)
    plt.figure(figsize=(6, 4))
    sns.heatmap(cm, annot=True, fmt='d', cmap='Blues', 
                xticklabels=['Ham', 'Spam'],
                yticklabels=['Ham', 'Spam'])
    plt.xlabel('Predicho')
    plt.ylabel('Real')
    plt.title('Matriz de Confusión')
    plt.tight_layout()
    plt.show()
    
    # Función para probar con nuevos correos
    def predecir_spam(nuevo_correo):
        correo_limpio = preprocesar_texto([nuevo_correo])
        X_nuevo = vectorizador.transform(correo_limpio)
        proba = clasificador.predict_proba(X_nuevo)[0]
        
        print(f"\nCorreo: {nuevo_correo}")
        print(f"Probabilidad de ser spam: {proba[1]*100:.2f}%")
        print(f"Clasificación: {'Spam' if proba[1] > 0.5 else 'Ham'}")
    
    # Probar con algunos ejemplos
    print("\n=== Probando el Filtro de Spam ===")
    predecir_spam("Hola, ¿cómo estás? ¿Podemos vernos mañana?")
    predecir_spam("¡Gana dinero fácil desde casa!")
    predecir_spam("Reunión de equipo mañana a las 10 AM")
    predecir_spam("¡Oferta exclusiva solo por hoy!")

Ejercicio 1: Aplicación del Teorema de Bayes

Resuelve los siguientes problemas utilizando el Teorema de Bayes:

  1. En una fábrica, las máquinas A, B y C producen el 45%, 30% y 25% de los artículos respectivamente. Los porcentajes de artículos defectuosos producidos por cada máquina son 3%, 4% y 5% respectivamente. Si se selecciona un artículo al azar y resulta defectuoso, ¿cuál es la probabilidad de que haya sido producido por la máquina B?
  2. Una prueba de detección de una enfermedad tiene una sensibilidad del 95% (probabilidad de dar positivo si tiene la enfermedad) y una especificidad del 90% (probabilidad de dar negativo si no tiene la enfermedad). Si la prevalencia de la enfermedad en la población es del 2%, ¿cuál es la probabilidad de que una persona que dio positivo en la prueba realmente tenga la enfermedad?
def ejercicio1():
    """
    Solución al ejercicio 1
    """
    # Datos del problema 1
    # Probabilidades a priori
    p_a = 0.45  # Máquina A
    p_b = 0.30  # Máquina B
    p_c = 0.25  # Máquina C
    
    # Probabilidades de defectuosos
    p_dado_a = 0.03  # P(Defectuoso|A)
    p_dado_b = 0.04  # P(Defectuoso|B)
    p_dado_c = 0.05  # P(Defectuoso|C)
    
    # Probabilidad total de que un artículo sea defectuoso
    p_d = p_a * p_dado_a + p_b * p_dado_b + p_c * p_dado_c
    
    # Aplicar Teorema de Bayes para P(B|Defectuoso)
    p_b_dado_d = (p_dado_b * p_b) / p_d
    
    print("=== Solución al Problema 1 ===")
    print(f"Probabilidad de que un artículo defectuoso sea de la máquina B: {p_b_dado_d*100:.2f}%")
    
    # Datos del problema 2
    sensibilidad = 0.95  # P(Prueba+|Enfermo)
    especificidad = 0.90  # P(Prueba-|Sano)
    prevalencia = 0.02    # P(Enfermo)
    
    # P(Enfermo|Prueba+) = P(Prueba+|Enfermo) * P(Enfermo) / P(Prueba+)
    p_prueba_positivo = sensibilidad * prevalencia + (1 - especificidad) * (1 - prevalencia)
    p_enfermo_dado_positivo = (sensibilidad * prevalencia) / p_prueba_positivo
    
    print("\n=== Solución al Problema 2 ===")
    print(f"Probabilidad de tener la enfermedad dado positivo: {p_enfermo_dado_positivo*100:.2f}%")

ejercicio1()

def preprocesar_texto(texto):
    """
    Preprocesa el texto para el análisis
    """
    # Convertir a minúsculas
    texto = texto.lower()
    
    # Eliminar puntuación
    texto = texto.translate(str.maketrans('', '', string.punctuation))
    
    # Tokenización
    tokens = nltk.word_tokenize(texto)
    
    # Eliminar stopwords
    stop_words = set(stopwords.words('english'))
    tokens = [word for word in tokens if word not in stop_words]
    
    # Aplicar stemming
    stemmer = PorterStemmer()
    tokens = [stemmer.stem(word) for word in tokens]
    
    # Unir tokens de nuevo en texto
    return ' '.join(tokens)

def entrenar_modelo(df):
    """
    Entrena un modelo de clasificación de texto
    """
    # Preprocesar texto
    df['texto_limpio'] = df['text'].apply(preprocesar_texto)
    
    # Dividir en conjuntos de entrenamiento y prueba
    X_train, X_test, y_train, y_test = train_test_split(
        df['texto_limpio'], 
        df['label'], 
        test_size=0.3, 
        random_state=42,
        stratify=df['label']
    )
    
    # Vectorizar texto con TF-IDF
    vectorizador = TfidfVectorizer(max_features=5000)
    X_train_vec = vectorizador.fit_transform(X_train)
    X_test_vec = vectorizador.transform(X_test)
    
    # Entrenar modelo Naive Bayes
    modelo = MultinomialNB()
    modelo.fit(X_train_vec, y_train)
    
    return modelo, vectorizador, X_test_vec, y_test

def evaluar_modelo(modelo, X_test, y_test):
    """
    Evalúa el rendimiento del modelo
    """
    # Hacer predicciones
    y_pred = modelo.predict(X_test)
    
    # Mostrar métricas
    print("\n=== Métricas de Evaluación ===")
    print(classification_report(y_test, y_pred))
    
    # Mostrar matriz de confusión
    cm = confusion_matrix(y_test, y_pred)
    plt.figure(figsize=(6, 4))
    sns.heatmap(cm, annot=True, fmt='d', cmap='Blues', 
                xticklabels=modelo.classes_,
                yticklabels=modelo.classes_)
    plt.xlabel('Predicho')
    plt.ylabel('Real')
    plt.title('Matriz de Confusión')
    plt.tight_layout()
    plt.show()

def interfaz_usuario(modelo, vectorizador):
    """
    Interfaz simple para probar el modelo
    """
    print("\n=== Filtro de Spam - Prueba el Modelo ===")
    print("Escribe 'salir' para terminar")
    
    while True:
        mensaje = input("\nIngresa un mensaje para clasificar: ")
        
        if mensaje.lower() == 'salir':
            break
        
        # Preprocesar mensaje
        mensaje_limpio = preprocesar_texto(mensaje)
        
        # Vectorizar
        X = vectorizador.transform([mensaje_limpio])
        
        # Predecir
        probabilidad = modelo.predict_proba(X)[0]
        prediccion = modelo.predict(X)[0]
        
        # Mostrar resultados
        print(f"\nMensaje: {mensaje}")
        print(f"Probabilidades: {dict(zip(modelo.classes_, probabilidad.round(4)))}")
        print(f"Clasificación: {prediccion}")

Resumen

  • El Teorema de Bayes es una herramienta fundamental para actualizar probabilidades a medida que se obtiene nueva información.
  • La probabilidad a priori representa nuestro conocimiento inicial sobre un evento, mientras que la probabilidad a posteriori es la probabilidad actualizada después de observar la evidencia.
  • El clasificador Naive Bayes es un algoritmo de aprendizaje automático basado en el Teorema de Bayes que asume independencia entre las características.
  • Naive Bayes es especialmente útil para clasificación de texto, como filtros de spam, análisis de sentimientos y categorización de documentos.
  • Aunque la suposición de independencia es raramente cierta en la práctica, Naive Bayes a menudo funciona sorprendentemente bien en muchos escenarios del mundo real.
  • La implementación de Naive Bayes es computacionalmente eficiente y requiere relativamente pocos datos de entrenamiento en comparación con otros algoritmos.

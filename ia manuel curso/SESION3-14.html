Introducción al Aprendizaje No Supervisado

¿Qué es el Aprendizaje No Supervisado?

El aprendizaje no supervisado es un tipo de aprendizaje automático donde el modelo se entrena con datos no etiquetados y debe encontrar patrones y relaciones por sí mismo.

¿Cuándo usarlo?

  • Cuando no hay etiquetas disponibles
  • Para explorar la estructura de los datos
  • Para reducción de dimensionalidad
  • Para detección de anomalías

Algoritmos de Aprendizaje No Supervisado

K-Means

Algoritmo de agrupamiento que divide los datos en K grupos basados en la distancia a los centroides.

  • Fácil de implementar
  • Eficiente en grandes conjuntos de datos
  • Sensible a la inicialización

DBSCAN

Agrupamiento basado en densidad de los puntos.

  • No requiere especificar el número de clusters
  • Encuentra clusters de forma arbitraria
  • Robusto a valores atípicos

PCA

Análisis de Componentes Principales para reducción de dimensionalidad.

  • Reduce la dimensionalidad
  • Elimina correlaciones
  • Útil para visualización

t-SNE

t-Distributed Stochastic Neighbor Embedding para visualización de alta dimensión.

  • Excelente para visualización
  • Mantiene estructuras locales
  • Computacionalmente costoso
from sklearn.cluster import KMeans
import numpy as np
import matplotlib.pyplot as plt

# Generar datos de ejemplo
np.random.seed(42)
X = np.random.randn(300, 2)

# Crear y ajustar el modelo K-Means
kmeans = KMeans(n_clusters=3, random_state=42)
kmeans.fit(X)

# Obtener las etiquetas y los centroides
labels = kmeans.labels_
centroids = kmeans.cluster_centers_

# Visualizar los resultados
plt.figure(figsize=(10, 6))
plt.scatter(X[:, 0], X[:, 1], c=labels, cmap='viridis', alpha=0.7)
plt.scatter(centroids[:, 0], centroids[:, 1], c='red', marker='X', s=200)
plt.title('Agrupamiento con K-Means')
plt.xlabel('Característica 1')
plt.ylabel('Característica 2')
plt.show()
from sklearn.cluster import DBSCAN
from sklearn.preprocessing import StandardScaler
import numpy as np
import matplotlib.pyplot as plt

# Generar datos de ejemplo
np.random.seed(42)
X = np.random.randn(300, 2)

# Estandarizar los datos
X = StandardScaler().fit_transform(X)

# Crear y ajustar el modelo DBSCAN
dbscan = DBSCAN(eps=0.3, min_samples=5)
labels = dbscan.fit_predict(X)

# Visualizar los resultados
plt.figure(figsize=(10, 6))
plt.scatter(X[:, 0], X[:, 1], c=labels, cmap='viridis')
plt.title('Agrupamiento con DBSCAN')
plt.xlabel('Característica 1 (estandarizada)')
plt.ylabel('Característica 2 (estandarizada)')
plt.show()
from sklearn.decomposition import PCA
from sklearn.datasets import load_iris
import matplotlib.pyplot as plt

# Cargar datos de ejemplo
iris = load_iris()
X = iris.data
y = iris.target

# Aplicar PCA
pca = PCA(n_components=2)
X_pca = pca.fit_transform(X)

# Visualizar los resultados
plt.figure(figsize=(10, 6))
scatter = plt.scatter(X_pca[:, 0], X_pca[:, 1], c=y, cmap='viridis')
plt.title('Reducción de dimensionalidad con PCA')
plt.xlabel('Primer componente principal')
plt.ylabel('Segunda componente principal')
plt.legend(handles=scatter.legend_elements()[0], 
           labels=iris.target_names,
           title="Clases")
plt.show()

Aplicaciones del Aprendizaje No Supervisado

Segmentación de Clientes

Agrupar clientes según su comportamiento de compra.

  • Personalización de ofertas
  • Estrategias de marketing dirigidas
  • Identificación de grupos de alto valor

Detección de Anomalías

Identificar patrones inusuales en los datos.

  • Detección de fraude
  • Monitoreo de equipos industriales
  • Seguridad informática

Reducción de Dimensionalidad

Simplificar conjuntos de datos complejos.

  • Visualización de datos
  • Eliminación de ruido
  • Preprocesamiento para otros algoritmos

Sistemas de Recomendación

Agrupar usuarios o productos similares.

  • Recomendaciones personalizadas
  • Descubrimiento de contenido
  • Análisis de mercado

Ejercicios Prácticos

Ejercicio 1: Agrupamiento de Clientes

Utiliza el conjunto de datos de clientes de un centro comercial:

  1. Carga y explora los datos
  2. Preprocesa los datos según sea necesario
  3. Aplica K-Means para segmentar a los clientes
  4. Visualiza los resultados
  5. Interpreta los grupos encontrados

Puedes usar el método del codo para encontrar el número óptimo de clusters.

Ejercicio 2: Reducción de Dimensionalidad

Utiliza un conjunto de datos de alta dimensión (por ejemplo, MNIST):

  1. Aplica PCA para reducir la dimensionalidad
  2. Visualiza los datos en 2D o 3D
  3. Compara con t-SNE
  4. Analiza la varianza explicada

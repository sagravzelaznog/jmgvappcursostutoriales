Introducción a los Operadores

Los operadores son símbolos especiales en Python que realizan operaciones en variables y valores. Python tiene varios tipos de operadores que nos permiten realizar diferentes tipos de operaciones.

Dato curioso: Python sigue las reglas matemáticas estándar de precedencia de operadores, comúnmente recordadas por el acrónimo PEMDAS (Paréntesis, Exponentes, Multiplicación y División, Suma y Resta).

1. Operadores Aritméticos

Se utilizan para realizar operaciones matemáticas básicas.

Operador Descripción Ejemplo Resultado
+ Suma 5 + 3 8
- Resta 10 - 4 6
* Multiplicación 6 * 7 42
/ División (siempre retorna float) 15 / 3 5.0
// División entera 15 // 4 3
% Módulo (resto de la división) 15 % 4 3
** Exponenciación 2 ** 3 8

Ejemplos prácticos:

# Operaciones básicas
suma = 10 + 5        # 15
resta = 20 - 8       # 12
multiplicacion = 6 * 7  # 42
division = 15 / 3     # 5.0 (siempre retorna float)
div_entera = 15 // 4  # 3 (división entera, trunca decimales)
resto = 15 % 4        # 3 (resto de la división)
potencia = 2 ** 3     # 8 (2 elevado a la 3)

2. Operadores de Comparación

Se utilizan para comparar valores y siempre retornan un valor booleano (True o False).

Operador Descripción Ejemplo Resultado
== Igual a 5 == 5 True
!= Diferente de 5 != 3 True
> Mayor que 10 > 5 True
< Menor que 10 < 5 False
>= Mayor o igual que 10 >= 10 True
<= Menor o igual que 5 <= 3 False

Ejemplos prácticos:

# Comparaciones
edad = 20
mayor_de_edad = edad >= 18  # True
igualdad = (10 == 10)       # True
desigualdad = (10 != 5)     # True

# Comparación de cadenas
nombre1 = "Ana"
nombre2 = "Beto"
orden_alfabetico = nombre1 < nombre2  # True (compara caracteres por su valor ASCII)

3. Operadores Lógicos

Se utilizan para combinar declaraciones condicionales.

Operador Descripción Ejemplo Resultado
and Devuelve True si ambas afirmaciones son verdaderas True and False False
or Devuelve True si al menos una afirmación es verdadera True or False True
not Invierte el resultado, devuelve False si el resultado es verdadero not True False

Tabla de verdad:

A B A and B A or B not A
True True True True False
True False False True False
False True False True True
False False False False True

Ejemplos prácticos:

# Operadores lógicos
edad = 25
tiene_licencia = True
puede_manejar = edad >= 18 and tiene_licencia  # True

# Combinando operadores
numero = 10
es_par = (numero % 2) == 0  # True

# Uso de paréntesis para claridad
resultado = (5 > 3) and (10 <= 20)  # True

# Operador not
es_verdadero = True
no_es_verdadero = not es_verdadero  # False

4. Precedencia de Operadores

Python sigue un orden específico al evaluar expresiones, conocido como precedencia de operadores. Aquí está el orden de mayor a menor precedencia:

  1. Paréntesis: () - Tienen la mayor precedencia
  2. Exponenciación: **
  3. Signo positivo/negativo: +x, -x
  4. Multiplicación, División, Módulo, División entera: *, /, %, //
  5. Suma y Resta: +, -
  6. Operadores de comparación: <=, <, >, >=
  7. Operadores de igualdad: ==, !=
  8. Operadores de asignación: =, +=, -=, etc.
  9. Operadores de identidad: is, is not
  10. Operadores de pertenencia: in, not in
  11. Operadores lógicos: not, and, or

Ejemplo de precedencia:

# Ejemplo de precedencia
resultado1 = 5 + 3 * 2      # 11 (primero la multiplicación)
resultado2 = (5 + 3) * 2    # 16 (los paréntesis tienen prioridad)

# Expresión compleja
resultado3 = 10 + 2 * 3 ** 2  # 28 (primero 3**2=9, luego 2*9=18, luego 10+18=28)

# Usando paréntesis para claridad
resultado_claro = 10 + (2 * (3 ** 2))  # 28

Consejo: Siempre usa paréntesis para hacer tus expresiones más claras, incluso cuando no sean estrictamente necesarios. Esto hace que tu código sea más legible y evita errores.

Actividad Práctica

1. Calculadora de Propinas

Crea un programa que calcule el monto total a pagar en un restaurante, incluyendo una propina del 15% sobre el total de la cuenta.

Pista
Usa una variable para el monto de la cuenta, calcula el 15% de ese monto y súmalo al total.

2. Verificador de Números

Escribe un programa que pida un número al usuario y determine si es positivo, negativo o cero, y si es par o impar.

Pista
Usa operadores de comparación para verificar si el número es mayor, menor o igual a cero, y el operador módulo (%) para verificar si es par o impar.

3. Calculadora de IMC

Crea un programa que calcule el Índice de Masa Corporal (IMC) y proporcione una interpretación básica.

Fórmula: IMC = peso (kg) / (altura (m) ** 2)

Interpretación:

  • IMC < 18.5: Bajo peso
  • 18.5 ≤ IMC < 25: Peso normal
  • 25 ≤ IMC < 30: Sobrepeso
  • IMC ≥ 30: Obesidad

Ejercicios Adicionales

Ejercicio 1: Calculadora de Edad

Escribe un programa que pida al usuario su año de nacimiento y calcule su edad actual.

Pista: Puedes usar el módulo datetime para obtener el año actual.

Ejercicio 2: Conversor de Temperatura

Crea un programa que convierta entre grados Celsius y Fahrenheit. El programa debe preguntar al usuario qué conversión desea realizar.

Fórmulas:

  • Celsius a Fahrenheit: (C × 9/5) + 32
  • Fahrenheit a Celsius: (F - 32) × 5/9

Ejercicio 3: Verificador de Año Bisiesto

Escribe un programa que determine si un año es bisiesto o no.

Un año es bisiesto si:

  1. Es divisible entre 4, pero no entre 100, o
  2. Es divisible entre 400

Ejemplo: 2000 fue bisiesto, 1900 no fue bisiesto, 2024 es bisiesto.

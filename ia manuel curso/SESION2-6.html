Contenido Teórico

Derivadas de Orden Superior

Las derivadas de orden superior son simplemente derivadas de derivadas. La segunda derivada es la derivada de la primera derivada, la tercera derivada es la derivada de la segunda derivada, y así sucesivamente.

\[ f''(x) = \frac{d}{dx}\left(\frac{df}{dx}\right) = \frac{d^2f}{dx^2} \]

Aplicación: En física, la primera derivada de la posición con respecto al tiempo es la velocidad, y la segunda derivada es la aceleración.

Ejemplo:

Para \( f(x) = x^3 + 2x^2 + x + 1 \):

\[ f'(x) = 3x^2 + 4x + 1 \] \[ f''(x) = 6x + 4 \] \[ f'''(x) = 6 \] \[ f^{(4)}(x) = 0 \]

Regla de la Cadena

La regla de la cadena nos permite derivar funciones compuestas. Si tenemos una función \( y = f(g(x)) \), entonces:

\[ \frac{dy}{dx} = f'(g(x)) \cdot g'(x) \]

Ejemplo:

Para \( y = \sin(x^2) \):

\[ \frac{dy}{dx} = \cos(x^2) \cdot 2x \]

Importante: La regla de la cadena es fundamental en el entrenamiento de redes neuronales, donde se utiliza en la retropropagación (backpropagation).

Regla de L'Hôpital

Esta regla es útil para evaluar límites indeterminados de la forma \( \frac{0}{0} \) o \( \frac{\infty}{\infty} \).

Si \( \lim_{x \to a} \frac{f(x)}{g(x)} \) es una forma indeterminada, entonces:

\[ \lim_{x \to a} \frac{f(x)}{g(x)} = \lim_{x \to a} \frac{f'(x)}{g'(x)} \]

Ejemplo:

Evaluar \( \lim_{x \to 0} \frac{\sin(x)}{x} \):

\[ \lim_{x \to 0} \frac{\sin(x)}{x} = \lim_{x \to 0} \frac{\cos(x)}{1} = 1 \]

Análisis de Funciones

Las derivadas nos permiten analizar el comportamiento de las funciones:

Característica Condición Interpretación
Máximo local \( f'(x) = 0 \) y \( f''(x) < 0 \) La función tiene un pico en este punto
Mínimo local \( f'(x) = 0 \) y \( f''(x) > 0 \) La función tiene un valle en este punto
Punto de inflexión \( f''(x) = 0 \) y cambia de signo La concavidad de la función cambia

Actividad Práctica

1. Cálculo de derivadas con SymPy

Vamos a calcular derivadas de orden superior y aplicar la regla de la cadena usando SymPy.

Código en Python:

from sympy import symbols, diff, sin, cos, exp, ln, simplify

# Definir la variable simbólica
x = symbols('x')

# Ejemplo 1: Derivadas de orden superior
f1 = x**4 - 3*x**3 + 2*x**2 - 5*x + 1
df1 = diff(f1, x)        # Primera derivada
d2f1 = diff(f1, x, 2)    # Segunda derivada
d3f1 = diff(f1, x, 3)    # Tercera derivada

print("1. Función:", f1)
print("   Primera derivada:", df1)
print("   Segunda derivada:", d2f1)
print("   Tercera derivada:", d3f1)

# Ejemplo 2: Regla de la cadena
f2 = sin(x**2 + 3*x)  # f(g(x)) donde g(x) = x² + 3x
df2 = diff(f2, x)
print("\n2. Función:", f2)
print("   Derivada usando regla de la cadena:", df2)

# Ejemplo 3: Regla de L'Hôpital
from sympy import limit, oo

# Definir la función
f3 = (exp(x) - 1 - x) / (x**2)

# Calcular el límite cuando x->0
lim_f3 = limit(f3, x, 0)
print("\n3. Límite usando L'Hôpital:")
print(f"   lim(x->0) (e^x - 1 - x)/x² = {lim_f3}")
Explicación del código

Este código muestra cómo calcular derivadas de orden superior usando diff(f, x, n) donde n es el orden de la derivada.

También demuestra la regla de la cadena en acción y cómo usar SymPy para evaluar límites que requieren la regla de L'Hôpital.

2. Análisis de Funciones

Vamos a analizar una función para encontrar sus puntos críticos, máximos, mínimos y puntos de inflexión.

import numpy as np
import matplotlib.pyplot as plt
from scipy.optimize import minimize_scalar
from scipy.misc import derivative

def f(x):
    return x**4 - 4*x**3 + 2

def df(x):
    return 4*x**3 - 12*x**2

def d2f(x):
    return 12*x**2 - 24*x

# Encontrar puntos críticos (donde f'(x) = 0)
from scipy.optimize import root
critical_points = root(df, x0=[1, 3]).x  # Valores iniciales para encontrar raíces
critical_points = np.sort(critical_points)  # Ordenar los puntos críticos

# Clasificar los puntos críticos
for x0 in critical_points:
    d2f_x0 = d2f(x0)
    if d2f_x0 > 0:
        point_type = "Mínimo local"
    elif d2f_x0 < 0:
        point_type = "Máximo local"
    else:
        point_type = "Punto de silla o punto de inflexión"
    print(f"Punto crítico en x = {x0:.4f}: {point_type}")

# Encontrar puntos de inflexión (donde f''(x) = 0)
inflection_points = root(d2f, x0=[0, 2]).x
inflection_points = np.sort(inflection_points)
print(f"\nPuntos de inflexión en x = {inflection_points[0]:.4f} y x = {inflection_points[1]:.4f}")

# Crear la gráfica
x_vals = np.linspace(-1, 4, 400)
y_vals = f(x_vals)

plt.figure(figsize=(12, 8))
plt.plot(x_vals, y_vals, label='f(x) = x⁴ - 4x³ + 2', linewidth=2)

# Marcar puntos críticos
for x0 in critical_points:
    y0 = f(x0)
    plt.scatter([x0], [y0], color='red', s=100, zorder=5)
    plt.annotate(f'({x0:.2f}, {y0:.2f})', 
                 (x0, y0), 
                 textcoords="offset points",
                 xytext=(0,10), 
                 ha='center')

# Marcar puntos de inflexión
for x0 in inflection_points:
    y0 = f(x0)
    plt.scatter([x0], [y0], color='green', s=100, marker='s', zorder=5)
    plt.annotate(f'Inf. ({x0:.2f}, {y0:.2f})', 
                 (x0, y0), 
                 textcoords="offset points",
                 xytext=(0,-15), 
                 ha='center')

# Configuración de la gráfica
plt.axhline(0, color='black', linewidth=0.5)
plt.axvline(0, color='black', linewidth=0.5)
plt.grid(True, linestyle='--', alpha=0.7)
plt.title('Análisis de la función f(x) = x⁴ - 4x³ + 2')
plt.xlabel('x')
plt.ylabel('f(x)')
plt.legend()
plt.show()

En esta visualización:

  • La curva azul representa la función f(x) = x⁴ - 4x³ + 2.
  • Los puntos rojos son los puntos críticos (donde f'(x) = 0).
  • Los cuadrados verdes son los puntos de inflexión (donde f''(x) = 0).
  • Las anotaciones muestran las coordenadas de estos puntos importantes.
  • Puedes modificar la función en el código para analizar otras funciones.

3. Optimización con Derivadas

Vamos a implementar el método de Newton para encontrar raíces de una función, que utiliza derivadas para converger más rápidamente que los métodos de bisección o secante.

import numpy as np
import matplotlib.pyplot as plt

def newton_method(f, df, x0, tol=1e-6, max_iter=100):
    """
    Método de Newton para encontrar raíces de f(x) = 0
    
    Parámetros:
    f -- función para la que buscamos raíces
    df -- derivada de la función f
    x0 -- estimación inicial
    tol -- tolerancia para la convergencia
    max_iter -- número máximo de iteraciones
    """
    print(f"\nMétodo de Newton para f(x) = 0")
    print("-" * 50)
    print(f"{'Iteración':<10} {'x':<15} {'f(x)':<15} 'Error'")
    print("-" * 50)
    
    x = x0
    for i in range(max_iter):
        fx = f(x)
        if abs(fx) < tol:
            print(f"Convergencia alcanzada en {i} iteraciones.")
            return x
        
        dfx = df(x)
        if dfx == 0:
            print("Derivada cero. No se puede continuar.")
            return None
            
        x_new = x - fx / dfx
        error = abs(x_new - x)
        
        print(f"{i:<10} {x:<15.8f} {fx:<15.8f} {error:.2e}")
        
        if error < tol:
            print(f"Convergencia alcanzada en {i+1} iteraciones.")
            return x_new
            
        x = x_new
    
    print("Número máximo de iteraciones alcanzado.")
    return x

# Definir una función y su derivada
def f(x):
    return x**3 - 2*x - 5  # Queremos resolver x³ - 2x - 5 = 0

def df(x):
    return 3*x**2 - 2

# Aplicar el método de Newton
root = newton_method(f, df, x0=2.0)
print(f"\nRaíz aproximada: x ≈ {root:.8f}")
print(f"f({root:.8f}) = {f(root):.8f}")

# Visualización
x_vals = np.linspace(1, 3, 400)
y_vals = f(x_vals)

plt.figure(figsize=(10, 6))
plt.plot(x_vals, y_vals, label='f(x) = x³ - 2x - 5')
plt.axhline(0, color='black', linewidth=0.5)
plt.scatter([root], [0], color='red', label=f'Raíz aproximada: x ≈ {root:.6f}')
plt.title('Método de Newton para encontrar raíces')
plt.xlabel('x')
plt.ylabel('f(x)')
plt.legend()
plt.grid(True, linestyle='--', alpha=0.7)
plt.show()
Relación con el aprendizaje automático

El método de Newton es un algoritmo de optimización que utiliza información de segundo orden (derivadas segundas) para converger más rápidamente a un mínimo o raíz. En el contexto del aprendizaje automático:

  • Es la base para métodos de optimización de segundo orden como el método de Newton-Raphson.
  • Es más rápido que el descenso de gradiente estándar pero puede ser más costoso computacionalmente para problemas de alta dimensión.
  • Se utiliza en algoritmos como el método de Newton-CG para regresión logística.

Resumen

  • Las derivadas de orden superior nos permiten analizar la concavidad y puntos de inflexión de funciones.
  • La regla de la cadena es esencial para derivar funciones compuestas y es fundamental en la retropropagación de redes neuronales.
  • La regla de L'Hôpital es útil para evaluar límites indeterminados.
  • El análisis de funciones con derivadas nos ayuda a encontrar máximos, mínimos y puntos de inflexión.
  • Los métodos numéricos como el de Newton utilizan derivadas para encontrar raíces y optimizar funciones de manera eficiente.

Contenido Teórico

¿Qué es Python?

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general. Fue creado por Guido van Rossum y lanzado por primera vez en 1991. Es conocido por su sintaxis clara y legible, lo que lo hace ideal para principiantes.

Dato curioso: El nombre "Python" proviene del grupo de comedia británico Monty Python, no de la serpiente, aunque el logo oficial sí muestra dos serpientes.

¿Por qué Python para IA?

  • Amplio ecosistema de bibliotecas como TensorFlow, PyTorch, y scikit-learn
  • Sintaxis legible que facilita la implementación de algoritmos complejos
  • Gran comunidad de desarrolladores
  • Multiplataforma (Windows, macOS, Linux)
  • Extensa documentación y recursos de aprendizaje

Actividad Práctica

1. Instalación de Python

  1. Visita python.org/downloads
  2. Haz clic en "Download Python 3.x.x" (la versión más reciente)
  3. Ejecuta el instalador descargado
  4. Asegúrate de marcar la casilla "Add Python to PATH"
  5. Haz clic en "Install Now"
  6. Verifica la instalación abriendo Command Prompt y escribiendo: python --version
  1. Abre Terminal (puedes encontrarlo en Aplicaciones > Utilidades)
  2. Instala Homebrew si no lo tienes: /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  3. Instala Python: brew install python
  4. Verifica la instalación: python3 --version

  2. Actualiza la lista de paquetes: sudo apt update
  3. Instala Python: sudo apt install python3 python3-pip
  4. Verifica la instalación: python3 --version

2. Instalación de Visual Studio Code

  1. Descarga VS Code desde code.visualstudio.com
  2. Ejecuta el instalador y sigue las instrucciones
  3. Una vez instalado, abre VS Code y ve a la pestaña de extensiones (Ctrl+Shift+X)
  4. Busca e instala la extensión "Python" de Microsoft

3. Tu primer programa en Python

  1. Abre VS Code
  2. Crea un nuevo archivo (Ctrl+N)
  3. Escribe el siguiente código: 
    # Mi primer programa en Python
print("¡Hola, Mundo!")

# Pide el nombre del usuario
nombre = input("¿Cuál es tu nombre? ")
print(f"¡Bienvenido al curso de IA, {nombre}!")
  4. Guarda el archivo como hola_mundo.py
  5. Ejecútalo haciendo clic derecho en el editor y seleccionando "Run Python File in Terminal" o presionando F5

4. Instalación de Jupyter Notebook

Jupyter Notebook es una herramienta interactiva ampliamente utilizada en ciencia de datos.

  1. Abre una terminal o Command Prompt
  2. Instala Jupyter Notebook: pip install notebook
  3. Inicia Jupyter Notebook: jupyter notebook
  4. Se abrirá una ventana en tu navegador predeterminado
  5. Haz clic en "New" > "Python 3" para crear un nuevo notebook

Ejercicios

Ejercicio 1: Calculadora Básica

Crea un programa que pida dos números al usuario y muestre la suma, resta, multiplicación y división.

Pista
Usa input() para obtener los números (recuerda convertirlos a float() o int()) y luego muestra los resultados con print().

Ejercicio 2: Conversor de Temperatura

Escribe un programa que convierta grados Celsius a Fahrenheit. La fórmula es: F = (C × 9/5) + 32.

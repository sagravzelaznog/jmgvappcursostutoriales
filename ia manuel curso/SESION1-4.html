Introducción a las Estructuras de Control

Las estructuras de control nos permiten controlar el flujo de ejecución de un programa, tomando decisiones y repitiendo acciones según sea necesario. Son fundamentales para crear programas dinámicos y flexibles.

Dato curioso: Las estructuras de control son la base de la lógica de programación y se utilizan en todos los lenguajes de programación, aunque la sintaxis puede variar.

1. Estructuras Condicionales

Permiten ejecutar bloques de código solo si se cumple cierta condición.

1.1 Sentencia if

Ejecuta un bloque de código si la condición es verdadera.

edad = 18
if edad >= 18:
    print("Eres mayor de edad")  # Esta línea se ejecuta si la condición es verdadera

1.2 Sentencia if-else

Ejecuta un bloque si la condición es verdadera y otro si es falsa.

edad = 16
if edad >= 18:
    print("Puedes votar")
else:
    print("Aún no puedes votar")  # Se ejecutará este bloque

1.3 Sentencia if-elif-else

Permite evaluar múltiples condiciones en secuencia.

nota = 85
if nota >= 90:
    print("Excelente (A)")
elif nota >= 80:
    print("Muy bueno (B)")  # Se ejecutará este bloque
elif nota >= 70:
    print("Bueno (C)")
else:
    print("Necesitas mejorar")

2. Bucles (Loops)

Permiten ejecutar un bloque de código múltiples veces.

2.1 Bucle for

Itera sobre una secuencia (lista, tupla, cadena, etc.).

# Iterar sobre una lista
frutas = ["manzana", "banana", "cereza"]
for fruta in frutas:
    print(f"Me gusta la {fruta}")

# Usando range()
for i in range(5):  # 0, 1, 2, 3, 4
    print(f"Número: {i}")

# Con inicio y fin
for i in range(2, 6):  # 2, 3, 4, 5
    print(i)

# Con paso
for i in range(0, 10, 2):  # 0, 2, 4, 6, 8
    print(i)

2.2 Bucle while

Ejecuta un bloque mientras una condición sea verdadera.

# Contador
contador = 0
while contador < 5:
    print(f"Contador: {contador}")
    contador += 1  # Incrementa el contador

# Bucle con condición de salida
respuesta = ""
while respuesta.lower() != "salir":
    respuesta = input("Escribe 'salir' para terminar: ")

3. Control de Bucles

Python proporciona palabras clave para controlar el flujo de los bucles.

3.1 break

Termina el bucle inmediatamente.

for i in range(10):
    if i == 5:
        break  # Sale del bucle cuando i es 5
    print(i)  # Imprime 0, 1, 2, 3, 4

3.2 continue

Salta a la siguiente iteración del bucle.

for i in range(5):
    if i == 2:
        continue  # Salta la iteración cuando i es 2
    print(i)  # Imprime 0, 1, 3, 4

3.3 else en bucles

Se ejecuta cuando el bucle termina normalmente (sin un break).

for i in range(3):
    print(i)
else:
    print("Bucle terminado")  # Siempre se ejecuta

for i in range(3):
    if i == 1:
        break
    print(i)
else:
    print("Esto no se ejecutará por el break")

4. Estructuras Anidadas

Podemos anidar estructuras de control para crear lógicas más complejas.

# Tabla de multiplicar del 1 al 5
for i in range(1, 6):
    print(f"\nTabla del {i}")
    print("-" * 12)
    
    for j in range(1, 11):
        print(f"{i} x {j:2} = {i*j:2}")
    
    if i < 5:
        print("\n" + "="*30 + "\n")

Ejercicio Práctico 3

Escribe un programa que determine si un número ingresado por el usuario es primo o no.

numero = int(input("Ingresa un número mayor que 1: "))

if numero <= 1:
    print("El número debe ser mayor que 1")
else:
    es_primo = True
    for i in range(2, int(numero ** 0.5) + 1):
        if numero % i == 0:
            es_primo = False
            break
    
    if es_primo:
        print(f"{numero} es un número primo")
    else:
        print(f"{numero} no es un número primo")

Proyecto Práctico

Calculadora de Calificaciones

Crea un programa que permita al usuario ingresar las calificaciones de varios estudiantes y calcule el promedio, la calificación más alta y la más baja.

Requisitos:

  1. Pregunta al usuario cuántos estudiantes hay en la clase.
  2. Para cada estudiante, pide su nombre y su calificación (0-100).
  3. Al final, muestra:
    • El promedio de la clase
    • La calificación más alta y el nombre del estudiante
    • La calificación más baja y el nombre del estudiante
    • Cuántos estudiantes aprobaron (nota ≥ 70)
# Inicialización de variables
num_estudiantes = int(input("¿Cuántos estudiantes hay en la clase? "))
estudiantes = []
suma_calificaciones = 0
aprobados = 0

# Validación de entrada
if num_estudiantes <= 0:
    print("El número de estudiantes debe ser mayor que cero.")
else:
    # Recopilación de datos
    for i in range(num_estudiantes):
        nombre = input(f"\nNombre del estudiante {i+1}: ")
        
        # Validación de la calificación
        while True:
            try:
                calificacion = float(input(f"Calificación de {nombre} (0-100): "))
                if 0 <= calificacion <= 100:
                    break
                else:
                    print("La calificación debe estar entre 0 y 100.")
            except ValueError:
                print("Por favor, ingresa un número válido.")
        
        # Guardar datos del estudiante
        estudiantes.append({
            'nombre': nombre,
            'calificacion': calificacion
        })
        
        # Actualizar estadísticas
        suma_calificaciones += calificacion
        if calificacion >= 70:
            aprobados += 1
    
    # Cálculos finales
    promedio = suma_calificaciones / num_estudiantes
    
    # Encontrar calificación más alta y más baja
    mejor_estudiante = max(estudiantes, key=lambda x: x['calificacion'])
    peor_estudiante = min(estudiantes, key=lambda x: x['calificacion'])
    
    # Mostrar resultados
    print("\n" + "="*50)
    print(f"{'RESUMEN DE CALIFICACIONES':^50}")
    print("="*50)
    print(f"Promedio de la clase: {promedio:.2f}")
    print(f"Mejor calificación: {mejor_estudiante['calificacion']} - {mejor_estudiante['nombre']}")
    print(f"Peor calificación: {peor_estudiante['calificacion']} - {peor_estudiante['nombre']}")
    print(f"Estudiantes aprobados: {aprobados} de {num_estudiantes} ({(aprobados/num_estudiantes)*100:.1f}%)")
    print("="*50)

Resumen de la Sesión

  • Las estructuras condicionales (if, elif, else) permiten tomar decisiones en el código.
  • Los bucles (for, while) permiten repetir bloques de código.
  • Las palabras clave break y continue controlan el flujo de los bucles.
  • Las estructuras de control se pueden anidar para crear lógicas más complejas.
  • Es importante validar las entradas del usuario para evitar errores.

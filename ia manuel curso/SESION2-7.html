Contenido Teórico

Derivadas Parciales

Una derivada parcial de una función de varias variables es su derivada con respecto a una de esas variables, manteniendo las demás como constantes.

Para una función \( f(x, y) \), las derivadas parciales se denotan como:

\[ \frac{\partial f}{\partial x} \quad \text{y} \quad \frac{\partial f}{\partial y} \]

Ejemplo:

Para \( f(x, y) = x^2y + y^3 \):

\[ \frac{\partial f}{\partial x} = 2xy \] \[ \frac{\partial f}{\partial y} = x^2 + 3y^2 \]

En aprendizaje automático: Las derivadas parciales son fundamentales para calcular cómo cambia la función de pérdida con respecto a cada parámetro del modelo.

Gradiente

El gradiente de una función escalar de varias variables es un vector que contiene todas sus derivadas parciales de primer orden.

\[ \nabla f(x, y) = \left( \frac{\partial f}{\partial x}, \frac{\partial f}{\partial y} \right) \]

Propiedades importantes:

  • El gradiente apunta en la dirección de máximo crecimiento de la función.
  • Su magnitud indica la tasa de crecimiento en esa dirección.
  • El gradiente es perpendicular a las curvas de nivel de la función.

Importante: En el descenso de gradiente, nos movemos en la dirección opuesta al gradiente para minimizar una función.

Regla de la Cadena para Derivadas Parciales

Para funciones compuestas, usamos la regla de la cadena. Si \( z = f(x, y) \) donde \( x = x(t) \) y \( y = y(t) \), entonces:

\[ \frac{dz}{dt} = \frac{\partial f}{\partial x} \frac{dx}{dt} + \frac{\partial f}{\partial y} \frac{dy}{dt} \]

Ejemplo:

Si \( z = x^2y \), \( x = \sin(t) \), \( y = t^2 \), entonces:

\[ \frac{dz}{dt} = (2xy)(\cos t) + (x^2)(2t) \] \[ = 2\sin(t) \cdot t^2 \cdot \cos(t) + \sin^2(t) \cdot 2t \]

Matriz Jacobiana

Para una función vectorial \( \mathbf{f}: \mathbb{R}^n \rightarrow \mathbb{R}^m \), la matriz jacobiana contiene todas las derivadas parciales de primer orden:

\[ J = \begin{bmatrix} \frac{\partial f_1}{\partial x_1} & \cdots & \frac{\partial f_1}{\partial x_n} \\ \vdots & \ddots & \vdots \\ \frac{\partial f_m}{\partial x_1} & \cdots & \frac{\partial f_m}{\partial x_n} \end{bmatrix} \]

Para una función escalar, el jacobiano es simplemente el vector gradiente transpuesto.

Actividad Práctica

1. Cálculo de derivadas parciales con SymPy

Vamos a calcular derivadas parciales y gradientes usando la biblioteca SymPy en Python.

Código en Python:

import sympy as sp

# Definir variables simbólicas
x, y = sp.symbols('x y')

# Definir una función de dos variables
f = x**2 * y + y**3 + sp.exp(x*y)

# Calcular derivadas parciales
df_dx = sp.diff(f, x)  # Derivada parcial con respecto a x
df_dy = sp.diff(f, y)  # Derivada parcial con respecto a y

print("Función:", f)
print("Derivada parcial con respecto a x:", df_dx)
print("Derivada parcial con respecto a y:", df_dy)

# Calcular el gradiente
gradient = [df_dx, df_dy]
print("\nGradiente:", gradient)

# Evaluar el gradiente en un punto específico
grad_at_point = [df_dx.subs({x: 1, y: 2}), df_dy.subs({x: 1, y: 2})]
print("\nGradiente en (1, 2):", grad_at_point)
Explicación del código

Este código utiliza SymPy para cálculo simbólico. Las funciones clave son:

  • sp.diff(f, x): Calcula la derivada de f con respecto a x.
  • expr.subs({x: 1, y: 2}): Evalúa la expresión en x=1, y=2.
  • El gradiente se construye como una lista de derivadas parciales.

2. Visualización de gradientes y curvas de nivel

Vamos a visualizar una función de dos variables, sus curvas de nivel y su campo de gradientes.

import numpy as np
import matplotlib.pyplot as plt

def f(x, y):
    return x**2 + y**2  # Función paraboloide

def grad_f(x, y):
    # Gradiente de f(x,y) = x² + y² es [2x, 2y]
    return 2*x, 2*y

# Crear una malla de puntos
x = np.linspace(-2, 2, 20)
y = np.linspace(-2, 2, 20)
X, Y = np.meshgrid(x, y)
Z = f(X, Y)

# Calcular el gradiente en cada punto
grad_x, grad_y = grad_f(X, Y)

# Normalizar los vectores para la visualización
magnitude = np.sqrt(grad_x**2 + grad_y**2)
grad_x = grad_x / (magnitude + 1e-8)  # Evitar división por cero
grad_y = grad_y / (magnitude + 1e-8)

# Crear la figura
plt.figure(figsize=(10, 8))

# Dibujar curvas de nivel
contour = plt.contour(X, Y, Z, levels=10, colors='black', alpha=0.5)
plt.clabel(contour, inline=True, fontsize=8)

# Dibujar el campo de gradientes
plt.quiver(X, Y, -grad_x, -grad_y, color='red', alpha=0.6, 
           width=0.003, scale=30, headwidth=3)

# Configuración de la gráfica
plt.title('Campo de gradientes y curvas de nivel de f(x,y) = x² + y²')
plt.xlabel('x')
plt.ylabel('y')
plt.axhline(0, color='black', linewidth=0.5)
plt.axvline(0, color='black', linewidth=0.5)
plt.grid(True, linestyle='--', alpha=0.7)
plt.axis('equal')
plt.colorbar(contour, label='f(x,y)')
plt.show()

En esta visualización:

  • Las líneas negras son curvas de nivel (todos los puntos en una misma curva tienen el mismo valor de f(x,y)).
  • Las flechas rojas representan el campo vectorial gradiente.
  • Observa que los vectores gradiente son perpendiculares a las curvas de nivel.
  • Los vectores apuntan en la dirección de máximo crecimiento de la función.

3. Implementación del descenso de gradiente

Vamos a implementar el algoritmo de descenso de gradiente para encontrar el mínimo de una función.

import numpy as np
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D

# Función a minimizar (paraboloide)
def f(x, y):
    return x**2 + 2*y**2 + 2*np.sin(2*np.pi*x)*np.cos(3*np.pi*y)

# Gradiente de la función
def grad_f(x, y):
    df_dx = 2*x + 4*np.pi*np.cos(2*np.pi*x)*np.cos(3*np.pi*y)
    df_dy = 4*y - 6*np.pi*np.sin(2*np.pi*x)*np.sin(3*np.pi*y)
    return np.array([df_dx, df_dy])

# Algoritmo de descenso de gradiente
def gradient_descent(start, learning_rate=0.01, max_iter=1000, tol=1e-6):
    # Inicializar
    x = np.array(start, dtype=float)
    path = [x.copy()]
    
    for i in range(max_iter):
        # Calcular el gradiente
        grad = grad_f(x[0], x[1])
        
        # Actualizar los parámetros
        x_new = x - learning_rate * grad
        path.append(x_new.copy())
        
        # Verificar convergencia
        if np.linalg.norm(x_new - x) < tol:
            print(f"Convergencia alcanzada en la iteración {i+1}")
            break
            
        x = x_new
    
    return np.array(path)

# Punto de inicio
start_point = [1.5, 1.5]

# Ejecutar el descenso de gradiente
path = gradient_descent(start_point, learning_rate=0.01, max_iter=1000)

# Crear una malla para la visualización
x = np.linspace(-2, 2, 100)
y = np.linspace(-2, 2, 100)
X, Y = np.meshgrid(x, y)
Z = f(X, Y)

# Crear la figura 3D
fig = plt.figure(figsize=(14, 6))

# Gráfica 3D
ax1 = fig.add_subplot(121, projection='3d')
surf = ax1.plot_surface(X, Y, Z, cmap='viridis', alpha=0.8)
ax1.plot(path[:, 0], path[:, 1], [f(p[0], p[1]) for p in path], 'r.-', markersize=10)
ax1.set_title('Descenso de gradiente en 3D')
ax1.set_xlabel('x')
ax1.set_ylabel('y')
ax1.set_zlabel('f(x,y)')

# Gráfica de contorno
ax2 = fig.add_subplot(122)
contour = ax2.contour(X, Y, Z, levels=20, cmap='viridis')
ax2.plot(path[:, 0], path[:, 1], 'r.-', markersize=10)
ax2.set_title('Trayectoria del descenso de gradiente')
ax2.set_xlabel('x')
ax2.set_ylabel('y')
plt.colorbar(contour, ax=ax2, label='f(x,y)')

plt.tight_layout()
plt.show()

# Mostrar el resultado final
final_point = path[-1]
print(f"Punto inicial: {start_point}, f(x,y) = {f(*start_point):.6f}")
print(f"Punto final:   {final_point}, f(x,y) = {f(*final_point):.6f}")
Relación con el aprendizaje automático

El descenso de gradiente es el algoritmo fundamental para entrenar modelos de aprendizaje automático:

  • En lugar de minimizar una función matemática, minimizamos una función de pérdida.
  • Los parámetros del modelo (pesos y sesgos) son las variables que ajustamos.
  • El gradiente nos dice cómo cambiar los parámetros para reducir la pérdida.
  • La tasa de aprendizaje controla el tamaño de los pasos que damos en cada iteración.
  • En la práctica, se usan variantes como SGD, Adam o RMSprop que mejoran la convergencia.

Resumen

  • Las derivadas parciales miden cómo cambia una función con respecto a una variable, manteniendo las demás constantes.
  • El gradiente es un vector que apunta en la dirección de máximo crecimiento de la función.
  • En el aprendizaje automático, el gradiente negativo apunta en la dirección de máximo descenso de la función de pérdida.
  • El descenso de gradiente es un algoritmo iterativo que utiliza el gradiente para encontrar mínimos de funciones.
  • La tasa de aprendizaje es un hiperparámetro crítico que controla el tamaño de los pasos del algoritmo.

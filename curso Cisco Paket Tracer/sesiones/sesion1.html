Introducción a las Redes de Computadoras

Objetivos de Aprendizaje

  • Comprender los conceptos básicos de redes de computadoras
  • Identificar los diferentes tipos de redes
  • Reconocer los componentes básicos de una red
  • Familiarizarse con la interfaz de Cisco Packet Tracer

¿Qué es una red de computadoras?

Una red de computadoras es un conjunto de equipos informáticos y dispositivos conectados entre sí para compartir información y recursos. Las redes permiten la comunicación entre dispositivos, el intercambio de datos y el acceso a recursos compartidos como impresoras, archivos e internet.

Tipos de redes

  • LAN (Red de Área Local): Cubre un área geográfica pequeña como una casa, oficina o edificio.
  • MAN (Red de Área Metropolitana): Cubre un área geográfica extensa como una ciudad.
  • WAN (Red de Área Amplia): Cubre grandes distancias geográficas, como países o continentes.

Componentes básicos de una red

  • Dispositivos finales: Computadoras, impresoras, teléfonos IP, etc.
  • Dispositivos de red: Routers, switches, hubs, puntos de acceso inalámbrico.
  • Medios de red: Cable de cobre, fibra óptica, conexión inalámbrica.
  • Protocolos: Conjunto de reglas que gobiernan las comunicaciones.

Introducción a Cisco Packet Tracer

Cisco Packet Tracer es un simulador de redes que permite crear topologías de red virtuales para practicar conceptos de redes sin necesidad de hardware físico.

Ejercicio 1: Explorando la Interfaz

Sigue estos pasos para familiarizarte con la interfaz de Packet Tracer:

  1. Abre Cisco Packet Tracer en tu computadora.
  2. Observa las diferentes áreas de la interfaz:
    • Barra de menú superior
    • Barra de herramientas principal
    • Área de trabajo central
    • Panel de dispositivos (abajo)
    • Panel de herramientas (derecha)
  3. Explora las diferentes categorías de dispositivos en el panel inferior.

La interfaz de Packet Tracer se divide en las siguientes áreas principales:

  1. Barra de menú: Contiene opciones como Archivo, Edición, Opciones, etc.
  2. Barra de herramientas: Contiene iconos para funciones comunes como guardar, abrir, etc.
  3. Área lógica/física: Permite cambiar entre la vista lógica y física de la red.
  4. Área de trabajo: Donde se construye y visualiza la red.
  5. Panel de dispositivos: Contiene todos los dispositivos que se pueden agregar a la red.
  6. Panel de herramientas: Contiene herramientas como selección, mover, eliminar, etc.

Ejercicio 2: Crear tu Primera Red

Crea una red simple siguiendo estos pasos:

  1. Arrastra dos computadoras desde el panel de dispositivos al área de trabajo.
  2. Arrastra un switch al área de trabajo.
  3. Conecta las computadoras al switch usando cables de red (selecciona el cable recto).
  4. Haz clic en cada computadora, ve a la pestaña "Desktop" y asigna direcciones IP a cada una en la misma red (por ejemplo, 192.168.1.1 y 192.168.1.2 con máscara 255.255.255.0).
  5. Intenta hacer ping de una computadora a la otra para verificar la conectividad.

Pasos detallados para verificar la conexión:

  1. Haz clic en una de las computadoras.
  2. Ve a la pestaña "Desktop".
  3. Abre la terminal de comandos.
  4. Escribe: ping 192.168.1.2 (o la IP de la otra computadora).
  5. Deberías ver respuestas exitosas si todo está configurado correctamente.

Si no funciona, verifica que:

  • Ambas computadoras estén en la misma red.
  • Los cables estén correctamente conectados (deben estar en verde).
  • Los dispositivos estén encendidos (el interruptor en la esquina superior derecha debe estar en posición "on").

Resumen de la Sesión

También has creado tu primera red simple y has verificado la conectividad entre dispositivos.

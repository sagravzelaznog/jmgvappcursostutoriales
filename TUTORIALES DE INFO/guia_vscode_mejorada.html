1. Configuración Inicial

Configura VS Code para obtener la mejor experiencia de desarrollo:

Idioma Español

  1. Abre VS Code
  2. Presiona Ctrl + Shift + X para abrir Extensiones
  3. Busca: "Spanish Language Pack"
  4. Haz clic en Install
  5. Reinicia el editor cuando te lo pida

Temas Recomendados

Personaliza la apariencia de tu editor con estos temas populares:

One Dark Pro
Tema oscuro con excelente contraste y colores vibrantes
Ctrl + K Ctrl + T
Dracula Official
Paleta de colores oscuros con tonos púrpuras
Ctrl + K Ctrl + T
Night Owl
Diseñado para programación nocturna
Ctrl + K Ctrl + T

Para cambiar rápidamente entre temas, presiona Ctrl + K Ctrl + T y selecciona tu tema favorito.

2. Configuración Avanzada (settings.json)

Personaliza VS Code a tu gusto editando el archivo de configuración:

  1. Presiona Ctrl + Shift + P
  2. Escribe: settings json
  3. Selecciona: "Preferences: Open User Settings (JSON)"
// Configuración recomendada para VS Code
// Archivo: settings.json
{
    "editor.tabSize": 2,
    "editor.fontSize": 14,
    "editor.wordWrap": "on",
    "editor.minimap.enabled": false,
    "files.autoSave": "afterDelay",
    "files.autoSaveDelay": 1000,
    "workbench.colorTheme": "One Dark Pro",
    "workbench.iconTheme": "material-icon-theme",
    "editor.bracketPairColorization.enabled": true,
    "emmet.includeLanguages": {
        "javascript": "javascriptreact",
        "vue-html": "html"
    },
    "editor.formatOnSave": true,
    "editor.defaultFormatter": "esbenp.prettier-vscode",
    "terminal.integrated.fontSize": 13,
    "explorer.confirmDelete": false,
    "explorer.confirmDragAndDrop": false,
    "workbench.startupEditor": "welcomePageInEmptyWorkbench",
    "javascript.updateImportsOnFileMove.enabled": "always",
    "typescript.updateImportsOnFileMove.enabled": "always"
}

Para que los cambios surtan efecto, guarda el archivo con Ctrl + S.

3. Atajos de Teclado Esenciales

Domina VS Code con estos atajos de teclado que te ahorrarán tiempo y mejorarán tu flujo de trabajo diario.

Domina VS Code con estos atajos de teclado imprescindibles:

Navegación Básica

Atajos para moverte rápidamente por tu código

Abrir archivo rápido
Ctrl + P
Busca y abre cualquier archivo en tu proyecto
Ir a la definición
F12
Navega a la definición de una función o variable
Ir al símbolo en el archivo
Ctrl + Shift + O
Busca símbolos en el archivo actual
Mostrar comandos
Ctrl + Shift + P
Accede a todos los comandos de VS Code

Edición Avanzada

Duplicar línea
Shift + Alt + ↓/↑
Duplica la línea actual o selección
Mover línea
Alt + ↑/↓
Mueve la línea actual hacia arriba o abajo
Comentar/Descomentar
Ctrl + /
Comenta o descomenta la línea o selección
Formatear documento
Shift + Alt + F
Formatea el documento actual

Multi-cursor

Añadir cursor
Alt + Click
Añade un cursor en la posición del clic
Seleccionar siguiente coincidencia
Ctrl + D
Selecciona la siguiente coincidencia de la selección actual
Seleccionar todas las coincidencias
Ctrl + Shift + L
Selecciona todas las coincidencias de la selección actual

Terminal Integrado

Abrir terminal
Ctrl + `
Abre o cierra el terminal integrado
Limpiar terminal
Ctrl + K
Limpia el contenido del terminal

5. Trucos y Consejos

Edición Rápida

  • Selección múltiple: Mantén presionado Alt y haz clic en diferentes posiciones para múltiples cursores
  • Seleccionar ocurrencias: Selecciona una palabra y presiona Ctrl + D para seleccionar la siguiente ocurrencia
  • Mover líneas: Alt + ↑/↓ para mover líneas hacia arriba o abajo
  • Duplicar línea: Shift + Alt + ↑/↓

Navegación Eficiente

  • Ir a línea: Ctrl + G y escribe el número de línea
  • Ir a definición: F12 o Ctrl + clic
  • Volver atrás: Alt + ←
  • Ir adelante: Alt + →

Atajos de Terminal

  • Abrir terminal: Ctrl + `
  • Nueva terminal: Ctrl + Shift + `
  • Limpiar terminal: Ctrl + K

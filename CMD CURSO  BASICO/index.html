⭐ Nivel: Principiante

1. Diagnóstico de Salud de la Batería Energía

+

Genera un reporte detallado sobre el estado de desgaste de la batería en laptops.

powercfg /batteryreport

Explicación: Crea un archivo .html en la carpeta de usuario con ciclos de carga y capacidad real vs original.

2. Reparación de Archivos del Sistema Sistema

+

Busca y repara archivos corruptos de Windows que causan pantallazos azules.

sfc /scannow
Nota: Requiere ejecutar CMD como Administrador.

3. Reparación de la Imagen de Windows (DISM) Sistema

+

Repara el almacén de componentes cuando SFC no puede solucionar los errores.

dism /online /cleanup-image /restorehealth

4. Verificación de Errores en Disco Disco

+

Escanea la superficie del disco en busca de sectores defectuosos.

chkdsk C: /f /r

Parámetros: /f corrige errores, /r recupera información legible de sectores dañados.

5. Reinicio de la Pila de Red (Winsock) Red

+

Soluciona problemas de "Conectividad Limitada" o DNS no responde.

netsh winsock reset

6. Liberación y Renovación de IP Red

+

Fuerza al router a asignar una nueva dirección IP al equipo.

ipconfig /release && ipconfig /renew

7. Limpieza de Caché DNS Red

+

Resuelve problemas al acceder a sitios web que cambiaron su IP.

ipconfig /flushdns

8. Obtener el Número de Serie (Hardware) Hardware

+

Identifica el equipo para garantía sin buscar la etiqueta física.

wmic bios get serialnumber

9. Listado de Controladores Instalados Drivers

+

Muestra todos los drivers y si están funcionando correctamente.

driverquery /v

10. Forzar Cierre de Programa Bloqueado Procesos

+

Cierra una aplicación que no responde cuando el Administrador de Tareas falla.

taskkill /f /im nombre_proceso.exe

Ejemplo: taskkill /f /im chrome.exe

11. Ver Conexiones de Red Activas Red

+

Detecta malware que se conecta a internet sin permiso.

netstat -ano

Muestra la IP remota y el PID (ID del proceso).

12. Reparación de Atributos de Carpeta Archivos

+

Recupera carpetas ocultas por virus en memorias USB.

attrib -s -h -r /s /d *.*

13. Diagnóstico de Información del Sistema Sistema

+

Reporte completo de hardware, versión de BIOS y parches de seguridad.

systeminfo

14. Gestión de Discos y Particiones Disco

+

Entrar a la consola avanzada de gestión de almacenamiento.

diskpart
Peligro: Comandos dentro de DISKPART pueden borrar datos permanentemente.

15. Obtener Dirección MAC Física Red

+

Necesario para configurar filtrado MAC en routers.

getmac /v

16. Apagado Programado Config

+

Útil para dejar una PC descargando o escaneando y que se apague sola.

shutdown -s -t 3600

3600 representa los segundos (1 hora).

17. Ver Modelo de la Placa Base Hardware

+

Saber qué Motherboard tiene la PC para descargar drivers específicos.

wmic baseboard get product,Manufacturer,version

18. Verificación de Asociaciones de Archivos Sistema

+

Repara problemas cuando un archivo (ej. .exe) no abre con el programa correcto.

assoc .exe

19. Comprobar Tiempo de Actividad (Uptime) Hardware

+

Verifica si el cliente realmente reinició la PC (útil en soporte remoto).

net statistics workstation | find "since"

20. Borrado Seguro de Espacio Libre Seguridad

+

Sobrescribe el espacio vacío del disco para que archivos borrados no sean recuperables.

cipher /w:C:

Solo afecta al espacio libre, no borra tus archivos actuales.

🔐 Sección del Maestro

Introduce el código para ver notas pedagógicas:

Estrategia Andragógica: Los ejercicios 4, 14 y 20 son críticos. Explicar siempre la diferencia entre comandos de solo lectura e intrusivos.

Tip de Reparación: Si `SFC` falla, siempre correr `DISM` con internet activo antes de intentar un segundo SFC.

Laboratorio Físico: Pide a los alumnos que generen el Battery Report (Ejer. 1) y analicen el desgaste en porcentajes.

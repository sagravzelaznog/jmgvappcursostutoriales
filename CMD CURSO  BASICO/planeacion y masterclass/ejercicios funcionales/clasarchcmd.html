💻 Masterclass: Magia en la Terminal

Aprende a clasificar miles de archivos en segundos usando CMD.

1. El Problema: El Caos Digital

Imagina una carpeta de "Descargas" con cientos de archivos: PDFs, imágenes, documentos de Word y ejecutables. Organizarlos a mano arrastrando y soltando tomaría horas.

👨‍🏫 Tip para el Profesor: Antes de mostrar el código, pide a los alumnos que abran su carpeta de descargas. Pregúntales cuánto tiempo les tomaría organizar 500 archivos manualmente. Esto activa el aprendizaje basado en problemas (Andragogía).

2. La Solución: El Bucle FOR en CMD

Windows tiene una herramienta poderosa llamada Símbolo del Sistema (CMD). Podemos escribir una sola línea de código que revise cada archivo, cree una carpeta con el nombre de su extensión y lo mueva allí.

for %i in (*.*) do mkdir "%~xi" & move "%i" "%~xi"
👨‍🏫 Tip para el Profesor: Advierte a los alumnos que NO ejecuten este comando en su escritorio principal todavía. Haz que creen una carpeta llamada "PruebaCMD", copien 5 archivos de diferentes tipos allí y abran el CMD específicamente en esa ruta.

3. Anatomía del Código

  • for %i in (*.*) : Inicia un bucle que selecciona absolutamente todos los archivos (*.*) en la carpeta actual y los guarda temporalmente en la variable %i.
  • do : Le dice al sistema "haz lo siguiente con cada archivo encontrado".
  • mkdir "%~xi" : Crea un directorio (carpeta). %~xi es la magia: extrae únicamente la extensión del archivo (por ejemplo, .pdf).
  • & : Permite ejecutar un segundo comando en la misma línea.
  • move "%i" "%~xi" : Mueve el archivo original (%i) hacia la nueva carpeta recién creada (%~xi).
👨‍🏫 Tip para el Profesor: Explica que al ejecutar esto, CMD mostrará un error de "El subdirectorio o archivo ya existe" si la carpeta ya fue creada por un archivo anterior. Aclara que es normal y el script seguirá funcionando (para evitar el error se usaría un condicional 'if not exist', pero para esta clase básica, mantener el código corto es mejor).

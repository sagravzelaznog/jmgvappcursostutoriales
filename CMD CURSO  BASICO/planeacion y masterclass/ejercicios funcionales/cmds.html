💻 Masterclass: Domina la Terminal

Aprende a mover y organizar archivos masivamente usando CMD.

1. El Reto: Archivos Desordenados

Todos tenemos esa carpeta llena de documentos, imágenes y descargas mezcladas. En lugar de arrastrar uno por uno, aprenderemos a pedirle a la computadora que mueva todos los archivos de un mismo tipo a una carpeta con el nombre que nosotros decidamos.

👨‍🏫 Tip para el Profesor: (Andragogía) Comienza preguntando a los alumnos cuántos archivos tienen en su carpeta de "Descargas". El dolor de tener que organizarlos manualmente creará la motivación intrínseca para aprender el comando.

2. La Fórmua Mágica: El comando MOVE

Usaremos el Símbolo del Sistema (CMD). El comando para lograr nuestro objetivo sigue esta estructura básica:

move *.extension "Nombre de Carpeta"
👨‍🏫 Tip para el Profesor: Antes de que ejecuten el comando, asegúrate de pedirles que creen una carpeta de prueba ("LaboratorioCMD") con algunos archivos de mentira. ¡Nunca se debe practicar por primera vez en carpetas de sistema!

3. Desglose del Código y Ejemplos

A. move (La Acción)

Es la orden directa al sistema para trasladar uno o varios elementos de un lugar a otro.

  • move archivo.txt Carpeta (Mueve un archivo específico)
  • move /Y *.jpg Fotos (Mueve y sobreescribe sin preguntar usando /Y)
  • move /-Y *.pdf Tareas (Mueve pero te pregunta antes de reemplazar)

B. *.extension (El Origen / Comodín)

El asterisco (*) es un "comodín" que significa "TODO". Al juntarlo con un punto y una extensión, selecciona todos los archivos de ese tipo.

  • *.mp3 (Selecciona toda la música mp3)
  • reporte_*.xlsx (Selecciona archivos Excel que empiecen con "reporte_")
  • *.* (Peligro: Selecciona absolutamente TODOS los archivos)

C. "Nombre Asignado" (El Destino)

Es el lugar a donde irán los archivos. Si el nombre de la carpeta tiene espacios, debe ir entre comillas.

  • "Mis Documentos Importantes" (Uso de comillas por los espacios)
  • C:\Respaldos\Fotos (Enviando a una ruta absoluta específica)
  • ..\CarpetaAnterior (Subiendo un nivel en los directorios)
👨‍🏫 Tip para el Profesor: Recuérdales que la carpeta de destino ya debe existir. Si no existe, pueden crearla rápidamente escribiendo mkdir "Nombre Asignado" justo antes de usar el comando move.
